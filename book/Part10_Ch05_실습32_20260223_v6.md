# 실습 ㉜: 세입자 거주 매도 의사결정 — 승계 vs 공실

세입자가 살고 있는 집을 팔 때 가장 골치 아픈 선택이 있습니다.
전세를 안고 갭투자자에게 팔 것인가(승계), 세입자를 내보내고 실입주자에게 팔 것인가(공실).
이 실습에서는 두 가지 시나리오의 예상 비용과 실수익(ROI)을 AI로 시뮬레이션하여, 데이터에 기반한 의사결정을 내립니다.

**학습 목표**

- 승계 매도와 공실 매도의 비용 구조 차이를 이해합니다
- AI 시뮬레이션으로 두 시나리오의 예상 실수익(ROI)을 비교합니다
- 세입자 협조 설계와 명도 협상 스크립트를 활용하여 실행 계획을 수립합니다

**핵심 키워드**
승계 매도 / 공실 매도 / 엑시트 수익률(ROI) / 보증금 반환 / 명도 위로금 / 세입자 협조 설계

---

## 승계와 공실, 수천만 원이 갈리는 선택이다

세입자가 거주 중인 매물의 매도는 두 가지 경로로 나뉩니다.

| 경로 | 대상 매수자 | 장점 | 단점 |
| ---- | ----------- | ---- | ---- |
| **승계 매도** (전세 안고 매도) | 갭투자자 | 보증금 반환 불필요, 즉시 매도 가능 | 갭투자 할인(수천만 원 네고) 불가피 |
| **공실 매도** (세입자 퇴거 후 매도) | 실입주자 | 시세대로 또는 프리미엄 가능 | 보증금 반환 자금 필요, 명도 리스크 |

이 선택은 단순히 편의의 문제가 아닙니다.
매도 후 내 통장에 들어오는 최종 실수익이 수천만 원까지 차이 날 수 있는 재무적 의사결정입니다.

승계 매도는 당장 돈이 들지 않아 편해 보이지만, 갭투자자는 반드시 할인을 요구합니다.
반면 공실 매도는 실입주자에게 높은 가격을 받을 수 있지만, 세입자에게 보증금을 먼저 돌려줘야 하므로 단기 대출 이자와 명도 위로금(이사비)이 발생합니다.

이 프롬프트는 두 가지 눈에 보이지 않는 비용을 저울질하여, 어느 쪽이 내게 더 많은 돈을 남겨주는지 수학적으로 비교합니다.

> ❓ **궁금해요 명도 위로금이란?**
>
> **명도 위로금** — 세입자에게 퇴거를 요청할 때 이사비 명목으로 지급하는 금액입니다.
> 법적 의무는 아니지만, 원만한 퇴거와 쇼잉 협조를 위해 관행적으로 200~500만 원 수준에서 협의합니다.
> 이 비용은 공실 매도의 숨은 비용이지만, 빈집으로 만들어 높은 가격에 팔면 충분히 회수 가능합니다.

---

## 프롬프트로 승계 vs 공실 시뮬레이션하기

이 실습은 세입자 거주 전용이므로 STEP 2 폼이 하나만 존재합니다.
1단계에서 AI에게 임대차 포함 매도 의사결정 분석가 역할을 부여하고, 2단계에서 임대차·재무·시장 정보를 입력합니다.

### 프롬프트를 보내기 전에

이 프롬프트의 평가 프레임은 **6가지 요인**으로 구성됩니다.

| 요인 | 평가 내용 |
| ---- | --------- |
| 만기-잔금 정합 | 임대차 만료일과 매도 잔금일의 시간 간격이 적정한가 |
| 보증금 반환 가능성 | 공실 매도 시 보증금을 돌려줄 자금 여력이 있는가 |
| 수요층 적합성 | 해당 지역·단지에 갭투자 수요와 실입주 수요 중 어느 쪽이 강한가 |
| 쇼잉 협조 난이도 | 세입자가 집 보여주기에 협조적인가 |
| 분쟁 리스크 | 갱신요구권, 보증금 분쟁 등의 가능성이 있는가 |
| 정책·대출 환경 | 현재 규제·금리 환경이 어느 쪽에 유리한가 |

6가지 요인을 종합하여 AI가 승계와 공실 각각의 예상 비용, 매도 기간, 실수익을 비교표로 출력합니다.

### 실행 방법

1. ChatGPT에서 새 대화창(New Chat)을 엽니다.
2. 최신모델을 선택하고, 웹 검색 기능을 켭니다.
3. 아래 1단계 프롬프트를 복사해서 붙여넣습니다.
4. AI가 확인 메시지를 출력하면 2단계 폼을 이어서 제출합니다.

💡 **Tip.** 승계 vs 공실 판단은 보증금 규모와 현금 여력에 크게 좌우됩니다. 내 통장 잔고와 추가 대출 가능 금액을 미리 확인해 두면 2단계 입력이 수월합니다.

### 프롬프트 — STEP 1: 공용 프롬프트

```
# [세입자 거주 매도 의사결정: 승계 vs 공실 | STEP 1]

[ROLE]
당신은 "임대차 포함 매도 의사결정 분석가"입니다.
목표는 승계 매도와 공실 매도 중 합리적 선택을 도출하는 것입니다.

[평가 6요인]
1) 만기-잔금 정합
2) 보증금 반환 가능성
3) 수요층 적합성
4) 쇼잉 협조 난이도
5) 분쟁 리스크
6) 정책·대출 환경

[입력 검증 게이트]
- 필수 항목 누락 시 분석 금지.

[OUTPUT]
1) 제약 요약
2) 승계 vs 공실 비교표
3) 추천 결론 + 전제조건
4) 실행 타임라인
5) 세입자 협조 설계
6) 실패 시 플랜B
7) 추가 질문
```

### 프롬프트 — STEP 2: 세입자 거주 입력 폼

```
# [승계 vs 공실 의사결정 | STEP 2 세입자 거주용]

※ [] 안의 문장은 입력 가이드입니다. 삭제하지 않아도 됩니다.

[필수]

- 분석 기준일(YYYY-MM-DD):
  [입력 가이드: 오늘 날짜
   예: 2026-03-15]

- 지역/단지명:
  [입력 가이드: 시·구 + 단지명
   예: 인천 연수구 송도 ○○아파트]

- 면적/타입:
  [입력 가이드: 평/전용 + 타입
   예: 33평 / 84타입]

- 매물 등록 시작일(대략):
  [입력 가이드: 노출 시작 시점
   예: 2025-10-01]

- 현재 호가:
  [입력 가이드: 현재 광고 가격
   예: 10.8억]

- 승계 가능 여부(가능/불가/협의):
  [입력 가이드: 세입자 승계로도 팔 수 있는지
   예: 승계 가능]

[임대차]

- 만료일(YYYY-MM-DD):
  [입력 가이드: 계약 만료일
   예: 2026-12-01]

- 보증금(대략) / 월세(있으면):
  [입력 가이드: 숫자로
   예: 보증금 4억 / 월세 0]

- 갱신요구권 이슈(있음/없음/모름):
  [입력 가이드: 알고 있는 만큼
   예: 아직 사용 안 함 / 모름]

- 쇼잉 협조 난이도(쉬움/보통/어려움):
  [입력 가이드: 집 보여주기 난이도
   예: 어려움(방문 자체를 꺼림)]

[최근 2~4주 반응]

- 조회/저장:
  [입력 가이드: 대략이라도
   예: 조회 900 / 저장 18]

- 문의:
  [입력 가이드: 문의 수
   예: 6건]

- 방문:
  [입력 가이드: 실제 방문 팀 수
   예: 1팀]

- 네고 요구:
  [입력 가이드: '요구 내용 + 반복'
   예: "공실 조건이면 살겠다" 요구가 대부분
   예: 가격 5천 인하 + 이사비 요구 1~2회]

[운영]

- 중개사 수:
  [입력 가이드: 내놓은 중개사 개수
   예: 3곳]

- 피드백 여부(예/아니오):
  [입력 가이드: 주간 리포트 받는지
   예: 아니오]

- 세입자 협조 비용 발생 여부:
  [입력 가이드: 협조/방문/정리 비용이 드는지
   예: 쇼잉 1회당 10만 원 요구
   예: 현재 비용 없음]

[본인 판단]

- 안 팔리는 이유(추정):
  [입력 가이드: 1~3개
   예: 승계 수요 부족 / 만기까지 너무 김 / 쇼잉이 어려움]

- 목표 우선순위:
  [입력 가이드: 하나 또는 순서
   예: 가격 > 리스크 최소 > 속도]

- 감당 가능한 조정 범위:
  [입력 가이드: 가격/조건 양보의 한계
   예: 최대 3천만 원 인하 가능
   예: 승계는 가능하나 보증금 조정은 불가]
```

### AI 응답 확인하기

폼을 제출하면 AI가 7개 섹션으로 승계 vs 공실 비교 분석을 출력합니다.
가장 먼저 확인해야 할 항목은 다음 3가지입니다.

| 순서 | 확인 항목              | 위치   | 왜 중요한가                                         |
| ---- | ---------------------- | ------ | --------------------------------------------------- |
| 1    | 승계 vs 공실 비교표    | 섹션 2 | 예상 비용, 매도 기간, 실수익이 한눈에 비교되는지 확인 |
| 2    | 추천 결론 + 전제조건   | 섹션 3 | AI가 어느 쪽을 추천하는지, 그 전제가 무엇인지 확인   |
| 3    | 세입자 협조 설계       | 섹션 5 | 퇴거 요청 또는 쇼잉 협조 스크립트가 실전에서 사용 가능한지 확인 |

AI가 공실 매도를 추천하더라도 명도 위로금(이사비)을 아까워하지 않는 것이 중요합니다.
200\~300만 원의 위로금으로 빈집을 만들면, 실입주자에게 매도 가격을 2,000\~3,000만 원 이상 올려 받을 수 있습니다.

> ⚠️ **주의:** AI의 시뮬레이션은 사용자가 입력한 데이터에 기반합니다. 보증금 반환 여력이나 대출 가능 금액이 부정확하면 결론이 달라질 수 있으므로, 은행 사전 상담을 통해 정확한 수치를 입력하는 것이 중요합니다.

### 확인할 포인트

| 확인 항목                            | 기준                                                 | 문제 시 대처                                              |
| ------------------------------------ | ---------------------------------------------------- | --------------------------------------------------------- |

## 핵심 정리

- **승계 vs 공실**: 세입자 거주 매도에서 가장 중요한 의사결정이며, 최종 실수익이 수천만 원 차이 날 수 있습니다.
- **6가지 평가 요인**: 만기 정합, 보증금 반환, 수요층, 쇼잉 협조, 분쟁 리스크, 정책 환경을 종합하여 판단합니다.
- **명도 위로금**: 공실 매도가 유리하다면 이사비를 아끼지 않습니다. 소액 투자로 매도가를 크게 올릴 수 있는 가장 효과적인 수단입니다.
- **데이터 정확도**: 보증금 반환 여력, 대출 가능 금액 등 재무 데이터가 정확할수록 시뮬레이션 결과의 신뢰도가 높아집니다.
