# Part 9. 매수 실전

Part 7에서 단지를 분석하고, Part 8에서 직접 현장을 확인했습니다.
이제 남은 것은 하나, 실제로 사는 것입니다.
이 Part에서는 적정가 산정부터 계약, 대출 실행, 등기 이전까지
매수의 전 과정을 AI와 함께 단계별로 진행합니다.

**학습 목표**
- 실거래가 데이터로 적정 매수가를 산정하고 협상 전략을 수립합니다
- 대출 규제를 반영한 자금 조달 계획서를 작성합니다
- 등기부등본 리스크를 점검하고 상황별 특약 조항을 생성합니다
- 계약일부터 등기까지 D-day 체크리스트를 완성합니다

**핵심 키워드**
적정 매수가 / 네고 전략 / 자금 조달 / 대출 시뮬레이션 / 등기부등본 / 특약 조항 / 법무사 / 등기 이전 / D-day 플래너

## 좋은 매물을 찾고도 놓치는 이유

Part 5에서 투자 프로필을 만들고, Part 6에서 유망 지역을 골랐습니다.
Part 7에서 단지를 비교 분석하고, Part 8에서 직접 임장까지 다녀왔습니다.
여기까지 완료한 사람은 전체 투자자 중 상위 10%에 해당합니다.

그런데 이 단계에서 포기하거나 실패하는 경우가 적지 않습니다.

'이 가격이 적정한 건지 모르겠어서 결국 안 샀다.'
'대출이 안 나올 줄 몰랐다. 계약금만 날릴 뻔했다.'
'등기부등본에 근저당이 잡혀 있었는데, 뭔지 몰라서 그냥 넘어갔다.'

분석 능력과 매수 실행 능력은 전혀 다른 영역입니다.
아무리 좋은 매물을 찾아도, 적정가를 모르면 비싸게 사게 됩니다.
계약서를 잘못 쓰면 수억 원의 손실로 이어질 수 있습니다.
대출이 잔금일까지 나오지 않으면 계약 자체가 파기됩니다.

이 Part에서는 AI를 활용하여 매수 실행의 네 가지 핵심 단계를 하나씩 진행합니다.

### 매수 프로세스 전체 흐름

아파트 매수는 크게 네 단계로 진행됩니다.

```
Part 8 산출물 (매수 후보 매물)
         │
         ↓
┌────────────────────────────────────┐
│ ① 적정가 산정    얼마에 살 것인가   │
│ → 실거래가 분석 + 네고 전략        │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ ② 자금 조달     돈을 어떻게 마련하나 │
│ → 대출 시뮬레이션 + 총비용 정산    │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ ③ 계약 체결     안전하게 계약하기    │
│ → 등기부등본 점검 + 특약 생성      │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ ④ 잔금·등기     내 이름으로 만들기   │
│ → 대출 실행 + 법무사 + 등기 이전   │
└────────────────┬───────────────────┘
                 ↓
         내 이름으로 등기 완료
```

각 단계에는 **AI가 도울 수 있는 영역**과 **직접 해야 하는 실무**가 함께 있습니다.

| 단계 | AI가 도울 수 있는 것 | 직접 해야 하는 것 |
|------|---------------------|-----------------|
| 적정가 산정 | 실거래가 분석, 네고 시나리오 | 공인중개사 방문, 매도인 협상 |
| 자금 조달 | 대출 가능액 계산, 비용 정산 | 은행 방문, 서류 제출, 대출 심사 |
| 계약 체결 | 등기부등본 해석, 특약 초안 | 계약서 서명, 계약금 송금 |
| 잔금·등기 | D-day 체크리스트, 일정 관리 | 은행·법무사·등기소 방문 |

AI는 데이터 분석과 문서 초안 작성에 강합니다.
하지만 은행에 대출을 신청하거나, 법무사에게 등기를 의뢰하거나, 계약서에 도장을 찍는 일은 직접 해야 합니다.
이 Part는 AI로 할 수 있는 것을 최대한 활용하되, 현장에서 직접 해야 할 일을 빠짐없이 안내합니다.

## 4개 실습의 흐름

Part 9의 전체 흐름입니다.
4개 실습이 매수 프로세스 순서대로 이어집니다.

```
Part 8 산출물 (매수 후보 매물 + MASTER PACK)
         │
         ↓
┌────────────────────────────────────┐
│ 실습 ㉜ 적정 매수가 산정과 네고 전략 │
│ → 적정가 범위 + 협상 시나리오       │
└────────────────┬───────────────────┘
                 │
    ─── 각 실습은 별도 대화창 ───
                 │
                 ↓
┌────────────────────────────────────┐
│ 실습 ㉝ 자금 조달 계획과 대출 시뮬   │
│ → 자금 조달 계획서 + 비용 정산표    │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ 실습 ㉞ 계약 전 리스크 점검과 특약   │
│ → 등기부등본 체크리스트 + 특약 4종   │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ 실습 ㉟ 대출 실행부터 등기까지 D-day │
│ → D-day 체크리스트 + 비용 정산      │
└────────────────┬───────────────────┘
                 ↓
         Part 10 매도 실전
```

### 실습별 개요

| 실습 | 제목 | 핵심 산출물 | 세션 |
|------|------|------------|------|
| ㉜ | 적정 매수가 산정과 네고 전략 | 적정가 범위 + 협상 시나리오 표 | 새 대화창 |
| ㉝ | 자금 조달 계획과 대출 시뮬레이션 | 자금 조달 계획서 + 월 상환액 | 새 대화창 |
| ㉞ | 계약 전 리스크 점검과 특약 생성 | 등기부등본 체크리스트 + 특약 4종 | 새 대화창 |
| ㉟ | 대출 실행부터 등기까지 D-day 플래너 | D-day 체크리스트 + 잔금일 타임테이블 | 새 대화창 |

Part 5~6과 달리, 각 실습은 **별도의 대화창**에서 진행합니다.
각 실습의 입력 조건이 다르고, 실습 ㉞에서는 등기부등본 원문을 붙여넣어야 하기 때문입니다.

다만 실습 ㉝의 매수 목표가는 실습 ㉜의 결과를 반영합니다.
실습 ㉟의 계약일과 잔금일은 실습 ㉞까지의 진행 상황을 반영합니다.
이전 실습의 핵심 결과를 다음 프롬프트의 [Context]에 넣어주면 됩니다.

> ⚠️ **주의:** 이 Part의 AI 프롬프트로 생성한 특약 조항, 계약서 검토, 세금 계산 결과는 참고 자료입니다.
> 실제 계약 시에는 반드시 공인중개사, 법무사, 세무사와 상담하세요.

## Part 8 산출물이 Part 9에서 쓰이는 방법

Part 8까지의 산출물이 Part 9 각 실습에서 어떻게 활용되는지 확인하세요.

| 이전 Part 산출물 | Part 9 활용 | 해당 실습 |
|-----------------|------------|----------|
| 매수 후보 매물 정보 (단지명, 면적, 층/향) | 프롬프트 [Context]에 입력 | ㉜ ㉝ ㉞ ㉟ 전체 |
| 매도 호가 | 적정가 산정의 기준점 | ㉜ |
| MASTER PACK (구매력, 투자유형) | 대출 시뮬레이션 입력값 | ㉝ |
| 임장 체크리스트 결과 | 하자 발견 시 특약 근거 | ㉞ |
| 유망 지역 리포트 (Part 6) | 지역 맥락 참고 | ㉜ ㉝ |

Part 5에서 "내가 얼마짜리 집을 살 수 있는가"를 계산했습니다.
Part 6에서 "어디에 투자할 것인가"를 정했습니다.
Part 7~8에서 "어떤 단지의 어떤 매물을 살 것인가"를 결정했습니다.

이 Part를 마치면 마지막 질문에 답할 수 있습니다.
"**어떻게 사야 안전하고, 얼마에 사야 적정한가.**"

등기부등본에 내 이름이 올라가는 그날까지, 빠짐없이 준비합시다.
이제 실습 ㉜에서 첫 번째 질문을 던져봅니다.
이 매물, 얼마에 사야 적정한가?

## 핵심 정리

- **4단계 매수 프로세스**: 적정가 산정 → 자금 조달 → 계약 체결 → 잔금·등기 순서로 진행됩니다. 한 단계라도 건너뛰면 수천만 원의 손실로 이어질 수 있습니다.
- **AI와 실무의 역할 구분**: AI는 데이터 분석과 문서 초안에 강하지만, 은행 방문·계약서 서명·등기 신청은 직접 해야 합니다. 각 단계에서 AI가 도울 수 있는 것과 직접 해야 하는 것을 구분하세요.
- **이전 Part 산출물 활용**: Part 5의 MASTER PACK(구매력, 투자유형)과 Part 8의 매수 후보 매물 정보가 Part 9 전체 실습의 입력값이 됩니다. 준비되지 않았다면 이전 Part를 먼저 완료하세요.
- **4개 실습은 순서대로**: 각 실습의 결과가 다음 실습의 입력이 됩니다. 실습 ㉜의 적정가가 실습 ㉝의 매매가가 되고, 실습 ㉞의 계약 조건이 실습 ㉟의 일정 기준이 됩니다.
