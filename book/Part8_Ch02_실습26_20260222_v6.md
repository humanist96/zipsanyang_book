# 실습 ㉖: 현장 임장 체크리스트 자동 생성

Part 7에서 분석 대상 단지와 협상 목표가를 잡았습니다. 이제 직접 현장에 가서 확인할 차례입니다.
하지만 아무 준비 없이 현장에 나가면 중요한 포인트를 놓치고 겉모습만 보고 돌아올 가능성이 높습니다.
이 실습에서는 AI가 Part 7 분석 결과를 바탕으로 해당 단지에 특화된 임장 체크리스트를 자동 생성합니다.

**학습 목표**

- Part 7 분석 결과 기반으로 해당 단지에 특화된 현장 확인 포인트를 도출합니다
- A4 출력용 체크리스트를 생성하여 현장에서 펜으로 직접 체크할 수 있도록 준비합니다
- 중개사에게 물어볼 구체적 질문 리스트를 작성합니다

**핵심 키워드**
임장 체크리스트 / 외부 점검 / 내부 점검 / 중개사 인터뷰 / 비교 평가표 / A4 출력

---

## 준비 없이 현장에 가면 후회만 남는다

처음 임장을 가면 "괜찮아 보인다", "별로다" 정도의 감만 갖고 돌아오기 쉽습니다.
그러다 계약 후에 "왜 그때 주차장을 안 봤지?", "왜 역까지 직접 안 걸어봤지?" 하며 후회하는 경우가 생깁니다.

실제로 흔한 사례가 있습니다. 주말 오후에 임장을 갔는데, 평일 출근 시간대의 교통 정체를 전혀 확인하지 못한 채 계약한 경우입니다.
또는 맑은 날에만 방문해서 비 오는 날 단지 입구가 침수되는 것을 알지 못한 경우도 있습니다.
이런 실수는 대부분 "무엇을 확인할지" 목록이 없었기 때문에 발생합니다.

**임장**<sup>현장을 직접 방문하여 확인하는 것</sup>에서 가장 중요한 것은 무엇을 확인할지 미리 정해두는 것입니다.
체크리스트가 있으면 놓치는 것이 없고, 없으면 감에 의존하게 됩니다.

이 체크리스트를 직접 만들 필요는 없습니다.
Part 7에서 분석한 내용을 AI에게 전달하면, 해당 단지에서 특히 확인해야 할 포인트가 반영된 맞춤형 체크리스트가 생성됩니다.
예를 들어 Part 7에서 "GTX 역까지 실제 거리 미확인"이 리스크로 도출되었다면, 체크리스트에 "GTX 예정 부지까지 실제 도보 시간 측정" 항목이 자동으로 추가됩니다.
일반적인 체크리스트에는 없는, 해당 단지에서만 확인해야 할 맞춤 항목이 포함된다는 점이 핵심입니다.

좋은 체크리스트는 4가지 섹션으로 구성됩니다.

| 섹션                   | 내용                                             |
| ---------------------- | ------------------------------------------------ |
| **외부/내부 현장 점검** | 역까지 실제 도보 시간, 소음, 채광, 주차 환경     |
| **중개사 인터뷰**       | 실제 거래 가능 가격, 급매 여부, 전세 수요        |
| **단지별 비교 평가표**  | 교통·쾌적성·생활 인프라·가격 매력·리스크 5점 채점 |
| **종합 의견**           | 최우선 단지, 보류 단지, 총평 기입                |

핸드폰으로 타이핑하는 것보다 **종이에 펜으로 체크하면서 눈으로 확인하는 것이 훨씬 꼼꼼합니다**.
현장에서 핸드폰을 꺼내 타이핑하면 눈이 화면에 고정되어 주변 환경을 놓치기 쉽습니다.
중개사가 말하는 내용을 바로 메모할 수도 있으므로, A4 출력을 권장합니다.

## AI로 맞춤형 임장 체크리스트 만들기

### 프롬프트를 보내기 전에

이 프롬프트는 Part 7 분석 결과(단지명, 적정가, 주요 리스크 등)를 입력하면, 해당 단지에 특화된 A4 출력용 체크리스트를 생성합니다.
일반적인 체크리스트가 아니라, Part 7에서 도출된 리스크가 체크 항목에 반영된다는 점이 핵심입니다.
따라서 Part 7 분석 결과를 가능한 한 구체적으로 입력할수록 체크리스트의 정밀도가 올라갑니다.

### 실행 방법

01. ChatGPT에서 새 대화창(New Chat)을 엽니다.
02. 아래 프롬프트를 복사해서 붙여넣습니다. 대괄호(`[ ]`) 안의 내용만 본인의 조건으로 바꿉니다.
03. AI가 4가지 섹션(외부/내부 점검, 중개사 인터뷰, 비교 평가표, 종합 의견)으로 구성된 체크리스트를 생성합니다.
04. 생성된 체크리스트를 A4로 출력합니다.

💡 **Tip.** 체크리스트를 출력할 때 클립보드에 끼워 가면 서서 메모하기 편리합니다. 펜은 여분으로 2개 준비하는 것을 권장합니다.

### 프롬프트

```
# Role (역할)

당신은 **현장 임장 전문 컨설턴트**입니다.
매수자가 현장에 들고 가서 펜으로 직접 체크하고 메모할 수 있는 **'A4 출력용 임장 체크리스트'**를 작성합니다.

**핵심 원칙**:
- 분석 데이터에서 도출된 리스크를 반드시 체크 항목에 반영
- 중개사에게 물어볼 질문은 구체적 숫자 기반으로 작성
- 흑백 인쇄 + 펜 필기에 최적화된 서식

---

# Mission (임무)

제공된 Part 7 분석 데이터를 바탕으로:

1. 해당 단지에 특화된 현장 확인 포인트 도출
2. 중개사 인터뷰 질문 리스트 작성 (구체적 숫자 포함)
3. 단지 비교 평가표 제공
4. 종합 의견 기입란 구성

# Critical Rules (핵심 규칙)

## 서식 규칙

1. **출력 최적화**: 이모지(Emoji) 사용 금지. 흑백 인쇄 시에도 깨지지 않는 기호(-, ■, □ 등)만 사용
2. **필기 공간 확보**: 모든 질문 아래에 밑줄(______________________)이나 2~3줄 여백 배치
3. **가독성**: 각 섹션 사이에 구분선(---) 삽입
4. **체크박스**: 모든 확인 항목 앞에 네모 체크박스([ ]) 배치

## 내용 규칙

5. **리스크 연동**: [사용자 입력]의 주요 리스크가 반드시 체크 항목에 포함되어야 함
6. **가격 질문 필수**: "실제 거래 가능한 바닥 가격"을 묻는 질문 반드시 포함 (호가 대비 괴리 확인)
7. **구체적 질문**: "좋아요?" 같은 모호한 질문 금지. "역까지 실제 도보 몇 분인가요?" 같은 수치형 질문으로 작성
8. **비교 단지 반영**: 비교 단지가 있으면 비교 평가표에 행 추가

---

# Output Format (출력 형식)

## [헤더] 임장 개요
- 방문 일시, 날씨, 동행인, 방문 단지명 등을 적을 수 있는 상단 정보란

## 1. 외부/내부 현장 점검 (Physical Check)
- 입지·접근성 / 단지 외부 환경 / 단지 내부 상품성 3개 하위 섹션으로 분류
- 각 항목 우측이나 아래에 [ 양호 / 보통 / 미흡 ] 체크 공간 + 메모란

## 2. 부동산 소장님 인터뷰 (Interview & Fact Check)
- 가격·매물 / 전세·수요 / 리스크 3개 하위 섹션으로 분류
- 각 질문에 구체적 숫자 기반 근거 명시 (예: "광고 호가 5.8억 vs 실거래 괴리 확인")
- 답변 메모용 밑줄 2~3줄 필수

## 3. 단지별 비교 평가표 (Scoring Table)
- 행: 단지명 / 열: 교통, 쾌적성, 생활 인프라, 가격 매력, 리스크, 총평
- 각 칸 빈칸 (현장에서 1~5점 기입)

## 4. 종합 의견 (Review)
- 최우선 검토 단지 / 보류·탈락 단지 / 최종 총평 / 추가 확인 필요 사항 기입란
```

### AI 응답 확인하기

프롬프트를 입력하면 AI가 4가지 섹션으로 구성된 체크리스트를 출력합니다.
의정부 민락푸르지오 84㎡를 기준으로 입력한 경우, 다음과 같은 형태의 결과가 나옵니다.

**입력 예시**

```
단지명: 민락푸르지오
지역: 경기도 의정부시 민락동
전용면적: 84㎡
적정가 범위: 5.4억 ~ 6.2억
협상 목표가: 5.5억
현재 호가: 5.8억
주요 리스크: GTX-C 민락역 2030년 예정(실제 거리 미확인), 주변 소음 미확인, 호가-실거래 차이 +5%, 전세가율 52%(갭투자 효율 낮음)
비교 단지: 없음
```

AI가 생성한 체크리스트에는 다음과 같은 항목이 포함됩니다.

- **입지·접근성 섹션**: "GTX-C 민락역 예정 부지까지 실제 도보 이동 시간(신호 대기 포함)" 항목이 자동으로 추가됩니다. 이것은 입력한 주요 리스크에 "GTX-C 실제 거리 미확인"이 있었기 때문입니다.
- **단지 외부 환경 섹션**: "주변 소음 수준" 항목이 강조됩니다. 리스크에 "소음 미확인"이 포함되어 있었기 때문입니다.
- **중개사 인터뷰 섹션**: "현재 실제 거래 가능한 바닥 가격은 얼마인가? (광고 호가 5.8억 vs 실거래 괴리 확인)"처럼 구체적 숫자가 포함된 질문이 생성됩니다.

각 항목에는 체크박스([ ])와 메모 공간(밑줄)이 함께 배치되어, 현장에서 펜으로 직접 기입할 수 있습니다.

핵심은 **입력한 리스크가 체크 항목에 반영되었는지** 확인하는 것입니다.
만약 리스크에 "전세가율 52%"를 넣었는데 체크리스트에 전세 관련 항목이 없다면, AI가 해당 리스크를 누락한 것이므로 "전세가율 관련 확인 항목을 추가해달라"고 재요청합니다.
비교 단지를 입력한 경우에는 비교 평가표에 해당 단지 행이 추가되어야 합니다. 행이 누락되었다면 마찬가지로 재요청합니다.

### 확인할 포인트

| 확인 항목                  | 기준                                   | 문제 시 대처                                  |
| -------------------------- | -------------------------------------- | --------------------------------------------- |
| 리스크가 체크 항목에 반영됨 | GTX 거리, 소음 등 입력 리스크 모두 포함 | 주요 리스크란에 구체적 리스크 3개 이상 입력    |
| 질문이 구체적임            | "실거래 가능 가격" 등 수치형 질문       | Critical Rules 전체가 복사되었는지 확인        |
| 메모 공간이 충분함         | 밑줄, 여백이 적절히 배치됨             | Output Format의 "메모란" 규칙 확인            |
| 흑백 출력에 문제없음       | 이모지 없이 □, ■, - 기호만 사용       | "이모지 사용 금지" 규칙이 포함되었는지 확인   |

> ⚠️ **주의:** AI가 입력 데이터의 리스크를 누락하고 일반적인 체크리스트만 생성하는 경우가 있습니다. GTX 거리, 소음, 호가 괴리 등 Part 7에서 도출된 리스크가 체크 항목에 포함되어 있는지 반드시 확인합니다.

## 체크리스트를 들고 현장으로 나가기

체크리스트가 완성되었으면 현장에 나가기 전에 몇 가지를 준비합니다.

| 준비 항목           | 설명                                        |
| -------------------- | ------------------------------------------- |
| **체크리스트 출력** | A4 2장 정도, 클립보드에 끼우면 편리함        |
| **펜 2개**          | 여분 필수                                   |
| **핸드폰 충전**     | 사진 촬영, AI 음성 대화용                   |
| **편한 신발**       | 역까지 직접 걸어볼 것이므로                 |

현장에서의 순서는 다음과 같습니다.

01. 단지 주변을 한 바퀴 걸으며 역/버스정류장까지 실제 도보 시간, 소음, 분위기를 확인합니다(10~15분).
02. 단지 내부에 들어가서 주차장, 조경, 커뮤니티 시설, 관리 상태를 확인합니다(10~15분).
03. 부동산에 방문하여 체크리스트의 질문 순서대로 물어보고 답변을 바로 메모합니다(20~30분).
04. 부동산을 나온 직후 종합 의견란에 총평을 작성합니다(5분). 돌아와서 쓰면 현장의 느낌이 희미해집니다.

한 가지 주의할 점이 있습니다. 가능하면 평일과 주말, 오전과 오후 등 **서로 다른 시간대에 2회 이상 방문**하는 것이 좋습니다.
주말 오후에는 조용했던 단지가 평일 출근 시간에는 교통 체증으로 소란스러울 수 있고, 맑은 날에는 보이지 않던 습기 문제가 비 온 뒤에 드러나기도 합니다.
시간 여유가 없다면 최소한 평일 오전에 1회 방문하는 것을 권장합니다. 실거주 환경을 가장 현실적으로 체감할 수 있는 시간대이기 때문입니다.

부동산에 방문할 때는 "○○동 84㎡ 매물 보러 왔는데요, 현재 실거래 가능한 가격이 얼마쯤인지 여쭤봐도 될까요?"라고 자연스럽게 시작하면 됩니다.
체크리스트에 질문이 다 적혀 있으므로 순서대로 물어보면 됩니다.
중개사와 대화할 때 체크리스트를 보는 것이 어색하게 느껴질 수 있지만, 실제로는 준비를 해온 매수자를 중개사가 더 진지하게 대합니다.
"꼼꼼하시네요"라는 반응이 나오면 오히려 좋은 신호입니다.

## 핵심 정리

- **맞춤형 체크리스트**: Part 7 분석 결과의 리스크가 자동으로 체크 항목에 반영되므로, 해당 단지에서 반드시 확인해야 할 포인트를 놓치지 않습니다.
- **A4 출력**: 현장에서 펜으로 체크하면서 확인하는 것이 핸드폰 타이핑보다 훨씬 꼼꼼합니다.
- **중개사 핵심 질문**: "실제 거래 가능한 바닥 가격은 얼마인가?"가 가장 중요한 질문입니다. 네이버 호가와 실제 거래 가능 가격은 다를 수 있습니다.
- **현장에서 바로 정리**: 종합 의견은 반드시 현장에서 작성합니다. 이 결과를 실습 ㉗에서 재평가의 입력값으로 사용합니다.
