# 실습 ㉞: 계약 전 리스크 점검과 특약 생성

적정가를 산정하고 자금 조달 계획을 세웠습니다.
계약서에 도장을 찍기 전에 한 가지 더 확인해야 합니다.
이 실습에서는 등기부등본의 위험 요소를 AI로 점검하고,
내 상황에 맞는 특약 조항을 생성합니다.

**학습 목표**
- 등기부등본의 갑구(소유권)와 을구(권리관계)를 해석합니다
- 근저당, 가압류, 가등기 등 위험 신호를 식별합니다
- 대출 미승인, 하자보수, 임차인 인수 등 상황별 특약 조항을 생성합니다

**핵심 키워드**
등기부등본 / 갑구 / 을구 / 근저당권 / 가압류 / 특약 조항 / 매매계약서 / 계약금 / 중도금 / 잔금

---

## 등기부등본은 매물의 건강검진서다

사람이 건강검진을 받듯, 아파트도 계약 전에 검진을 받아야 합니다.
그 검진서가 바로 **등기부등본**입니다.

**등기부등본** — 해당 부동산의 소유자, 면적, 권리관계(대출, 압류 등)가 모두 기록된 공적 문서입니다.
인터넷등기소(iros.go.kr)에서 누구나 1,000원에 발급받을 수 있습니다.

등기부등본은 크게 세 부분으로 나뉩니다.

| 구분 | 내용 | 확인 목적 |
|------|------|----------|
| **표제부** | 소재지, 면적, 구조 | 매물 정보가 정확한지 |
| **갑구** | 소유권 관련 사항 | 진짜 소유자가 맞는지, 압류가 없는지 |
| **을구** | 소유권 외 권리 | 대출(근저당)이 얼마나 잡혀 있는지 |

대부분의 매수자가 매매가와 위치만 확인하고 등기부등본은 대충 넘깁니다.
하지만 갑구에 **가압류**가 걸려 있으면 소유권 이전이 불가능할 수 있습니다.
을구에 **근저당 설정액**이 매매가를 초과하면 잔금을 치러도 깨끗한 소유권을 받지 못할 수 있습니다.

이런 항목을 하나하나 확인하는 것은 전문 지식이 필요합니다.
AI에게 등기부등본 내용을 입력하면, 위험 요소를 자동으로 분류하고 위험도를 평가해 줍니다.

> 🔴 **중요:** AI의 등기부등본 분석은 참고 자료입니다.
> 실제 계약 시에는 반드시 법무사 또는 변호사에게 전문 검토를 받으세요.
> 특히 가압류, 가등기, 신탁 등기가 있는 경우 전문가 상담은 필수입니다.

## 등기부등본 리스크 점검하기

AI에게 등기부등본 내용을 입력하고, 위험 요소 분석과 특약 조항 생성을 요청합니다.

### 실행 방법

**[준비]**
01. 인터넷등기소(iros.go.kr)에서 매수 후보 매물의 등기부등본을 발급받습니다. 열람용(700원) 또는 발급용(1,000원) 모두 가능합니다.
02. 등기부등본의 갑구와 을구 내용을 텍스트로 복사합니다.

**[실행]**
03. 새 대화창을 엽니다.
04. 아래 프롬프트의 [대괄호] 안을 채우고, 등기부등본 내용을 붙여넣습니다.
05. AI가 위험 요소 분석, 계약서 체크리스트, 특약 조항을 정리해 줍니다.

💡 **Tip.** 등기부등본을 아직 발급받지 않았다면, [Context]의 등기부등본 부분에 "미발급"이라고 입력하세요.
AI가 일반적인 체크리스트와 특약 조항을 먼저 제공하고, 발급 후 다시 분석을 요청하라고 안내합니다.

### 프롬프트

```
[Role]
당신은 부동산 거래 안전 전문가입니다.
등기부등본 분석과 매매계약서 검토 경력 10년 이상입니다.

[Mission]
아래 매물의 매매 계약 전 리스크를 점검하고,
내 상황에 맞는 특약 조항을 생성해주세요.

[Context]
- 대상 매물: [단지명], [동/호수]
- 매매가: [금액]억 원
- 계약금: [금액] (보통 매매가의 10%)
- 잔금 예정일: [날짜]
- 대출 예정 여부: [예/아니오]
- 입주 계획: [즉시 입주 / 전세 세팅]
- 현재 임차인: [있음(보증금 X억, 만기 YYYY.MM) / 없음]

등기부등본 내용 (갑구/을구를 아래에 붙여넣기):
[등기부등본 내용]

[Rules]
1. 등기부등본에서 반드시 확인해야 할 항목을 설명하세요:
   - 갑구: 소유권 이전 이력, 가압류, 가처분, 가등기, 경매 기입
   - 을구: 근저당권 설정액, 전세권, 임차권 등기
2. 각 위험 요소별 위험도를 🔴(높음) / 🟡(주의) / 🟢(안전)으로 표기하세요
3. 특약 조항은 실제 계약서에 넣을 수 있는 법률 문장으로 작성하세요
4. 법률적 판단이 필요한 사항은 반드시 "법무사 또는 변호사 상담이 필요합니다"를 명시하세요
5. 등기부등본이 없으면 "계약 전 반드시 등기부등본을 발급받아 확인하세요"를 안내하세요

[Output]
1. 등기부등본 체크리스트 (확인 항목 + 위험 신호 + 위험도)
2. 매매계약서 필수 확인 10개 항목 (표)
3. 상황별 특약 조항 4종:
   - 대출 미승인 시 계약 해제
   - 하자 발견 시 보수 비용 분담
   - 기존 임차인 인수 조건
   - 잔금일 등기 이전 지연 시 처리
4. 계약 당일 준비물 체크리스트
```

정대리의 경우 다음과 같이 입력합니다.

```
[Context]
- 대상 매물: 의왕 내손동 인덕원자이SK뷰, 103동 1204호
- 매매가: 5.5억 원
- 계약금: 5,500만 원 (10%)
- 잔금 예정일: 2026년 5월 15일
- 대출 예정 여부: 예 (약 2.2억 원)
- 입주 계획: 즉시 입주 (신혼집)
- 현재 임차인: 없음 (공실)

등기부등본 내용:
(발급 후 여기에 붙여넣기)
```

### AI 응답 확인하기

등기부등본이 깨끗한 경우(근저당만 있고 가압류 없음), AI는 다음과 같이 분석합니다.

**등기부등본 체크리스트**

| 확인 항목 | 위험 신호 | 위험도 | 정대리 매물 |
|----------|----------|--------|------------|
| 소유자 일치 | 매도인과 등기 소유자 불일치 | 🔴 | 🟢 일치 |
| 가압류 | 갑구에 가압류 등기 | 🔴 | 🟢 없음 |
| 가등기 | 갑구에 소유권이전청구권 가등기 | 🔴 | 🟢 없음 |
| 가처분 | 갑구에 처분금지 가처분 | 🔴 | 🟢 없음 |
| 경매 기입 | 갑구에 경매개시결정 | 🔴 | 🟢 없음 |
| 근저당 설정액 | 을구 근저당 합계가 매매가 초과 | 🟡 | 🟡 근저당 2.4억 (KB은행) |
| 전세권 설정 | 을구에 전세권 등기 | 🟡 | 🟢 없음 |
| 신탁 등기 | 갑구에 신탁 원부 기재 | 🟡 | 🟢 없음 |

근저당 2.4억은 매도인의 기존 주담대입니다.
잔금 수령 시 매도인이 근저당을 말소하는 것이 일반적인 절차입니다.
**특약에 "잔금일까지 근저당 말소 완료" 조건을 반드시 넣어야 합니다.**

> ❓ **궁금해요 근저당권이란?**
>
> 근저당권은 은행이 대출 담보로 설정하는 권리입니다.
> 매도인이 이 아파트를 담보로 대출을 받았다는 뜻입니다.
> 매매 시 잔금으로 매도인의 대출을 상환하고, 근저당을 말소한 뒤 깨끗한 소유권을 이전받습니다.
> 근저당 설정액이 매매가를 초과하면 위험 신호이므로, 법무사와 상담이 필요합니다.

**매매계약서 필수 확인 10개 항목**

| # | 확인 항목 | 확인 방법 |
|---|----------|----------|
| 1 | 매도인 신분증과 등기부 소유자 일치 | 신분증 실물 대조 |
| 2 | 등기부등본 발급일이 계약 당일인지 | 당일 발급본 확인 |
| 3 | 근저당권 말소 조건 | 특약에 명시 |
| 4 | 전입세대 확인서 | 동사무소 발급 |
| 5 | 계약금 금액과 지급 방법 | 계약서 명시 |
| 6 | 중도금 유무와 일정 | 계약서 명시 |
| 7 | 잔금일과 소유권 이전일 | 동일 날짜 확인 |
| 8 | 특약 사항 기재 여부 | 계약서 확인 |
| 9 | 중개수수료 금액과 지급 시점 | 중개사와 합의 |
| 10 | 매도인·매수인·중개사 서명 날인 | 계약 현장 확인 |

**상황별 특약 조항 4종**

AI가 생성한 특약 조항 예시입니다.
실제 사용 시 법무사의 검토를 받은 후 계약서에 반영하세요.

**① 대출 미승인 시 계약 해제**
```
매수인의 주택담보대출이 [은행명]으로부터 금 [금액]원 이상 승인되지 않을 경우,
매수인은 잔금 지급일 7일 전까지 서면으로 계약 해제를 통보할 수 있으며,
이 경우 매도인은 수령한 계약금 전액을 매수인에게 반환한다.
```

**② 하자 발견 시 보수 비용 분담**
```
잔금 지급일 이전에 매수인이 해당 부동산의 하자(누수, 곰팡이, 균열 등)를
발견한 경우, 매도인은 잔금 지급일까지 하자를 보수하거나,
보수 비용 상당액을 잔금에서 공제하기로 한다.
```

**③ 기존 임차인 인수 조건** (임차인이 있는 경우)
```
현재 임차인(보증금 [금액]원, 임대차 만기 [날짜])의 임대차 계약은
매수인이 그대로 승계하며, 매도인은 임차인에게 매수인 변경을 서면 통지한다.
임차인의 보증금 반환 의무는 잔금일 이후 매수인에게 이전된다.
```

**④ 잔금일 등기 이전 지연 시 처리**
```
매도인의 귀책사유로 잔금 지급일에 소유권 이전 등기에 필요한 서류가
제공되지 않을 경우, 매도인은 지연일수에 대해 잔금의 연 [N]%에 해당하는
지연손해금을 매수인에게 지급한다.
```

#### 확인할 포인트

| 항목 | 확인 내용 | 조치 |
|------|----------|------|
| 특약 법률 유효성 | AI 생성 특약이 법적으로 유효한 문장인가 | 법무사에게 검토 요청 |
| 근저당 말소 시점 | 잔금일 당일 말소 가능한지 | 매도인 측 은행에 확인 |
| 전입세대 확인 | 등기부에 없는 임차인이 있는지 | 동사무소 전입세대 확인서 |
| 계약금 비율 | 10%가 적정한지, 조정 가능한지 | 중개사와 협의 |

## 계약 당일 준비하기

계약서 서명은 한 번 하면 되돌리기 어렵습니다.
계약 당일에 필요한 준비물과 확인 순서를 정리합니다.

### 계약 당일 준비물

| 준비물 | 매수인 | 매도인 |
|--------|--------|--------|
| 신분증 | ✅ | ✅ |
| 도장 (인감 또는 서명) | ✅ | ✅ (인감도장 + 인감증명서) |
| 등기부등본 (당일 발급) | ✅ 확인용 | - |
| 계약금 송금 준비 | ✅ | - |
| 매매계약서 | 중개사 준비 | 중개사 준비 |
| 토지이용계획확인원 | 중개사 준비 | - |

### 계약 당일 순서

01. 등기부등본을 **당일 기준으로 다시 발급**받아 변동 사항이 없는지 확인합니다.
02. 매도인 신분증과 등기부 소유자가 일치하는지 확인합니다.
03. 계약서 내용(매매가, 계약금, 잔금일, 특약)을 꼼꼼히 읽습니다.
04. 특약 조항이 합의한 대로 기재되었는지 확인합니다.
05. 서명·날인 후 계약금을 매도인 계좌로 송금합니다.
06. 계약서 원본을 매수인·매도인·중개사가 각 1부씩 보관합니다.

> ⚠️ **주의:** 계약금을 송금할 때는 반드시 **매도인 본인 명의 계좌**로 보내세요.
> 중개사 계좌나 제3자 계좌로의 송금은 사기 위험이 있습니다.

## 핵심 정리

- **등기부등본은 필수**: 계약 전 반드시 등기부등본을 발급받아 갑구(소유권)와 을구(권리관계)를 확인하세요. 가압류, 가등기가 있으면 계약을 보류해야 합니다.
- **특약이 보험**: 대출 미승인 시 계약 해제, 하자 보수 분담 등 특약 조항은 매수인을 보호하는 안전장치입니다. AI로 초안을 만들고 법무사의 검토를 받으세요.
- **계약 당일 확인**: 등기부등본 당일 발급, 매도인 신분 확인, 특약 기재 확인, 계약금은 매도인 본인 계좌로 송금이 핵심입니다.
- **다음 단계**: 계약이 체결되면, 실습 ㉟에서 잔금일까지의 D-day 체크리스트를 만들고 대출 실행과 등기 이전을 준비합니다.
