# 실습 ㉕: 과거 수익률 분석

실습 ㉒에서 민락푸르지오를 1순위 단지로 선정하고, 실습 ㉓에서 적정가 범위를 확인했으며, 실습 ㉔에서 5.8억 타겟 매물을 잡았습니다.
이번 실습에서는 "이 단지를 2년 전에 샀으면 얼마나 벌었을까?"라는 질문을 수익률 데이터로 답합니다.
과거 수익률을 분석하면 투자 타이밍이 얼마나 중요한지 숫자로 확인할 수 있으며, 현재 매수 시점에 대한 근거 있는 판단을 세울 수 있습니다.
많은 분이 과거 수익률을 볼 때 미래 예측표처럼 읽습니다. 하지만 이 실습은 미래를 맞히는 것이 아니라, 같은 단지도 언제 샀느냐에 따라 결과가 얼마나 달라졌는지 확인하는 데 더 가깝습니다.

**학습 목표**

- 2단계 프롬프트 워크플로를 활용하여 과거 실거래 기반 수익률 분석 결과를 도출합니다
- 갭투자 수익률 계산 구조를 이해하고 결과표를 해석합니다
- 과거 환경과 현재 환경을 비교하여 현재 매수 시점의 적절성을 판단합니다

**핵심 키워드**
갭투자 / 실투자금 / 레버리지 효과 / 전세가율 / 2년 보유 수익률 / 시장 사이클 / 깡통전세

## 타이밍이 만드는 100%p 수익률 차이

Part 7의 실습 ㉒~㉔를 통해 어떤 단지를, 어떤 가격에 사야 하는지 데이터로 정리했습니다.
남은 질문은 하나입니다.

'지금이 사기에 좋은 시점인가?'

이 질문에 답하는 방법 중 하나가 과거 수익률 분석입니다.
과거 수익률은 앞유리를 보는 자료라기보다 백미러에 가깝습니다. 어디에서 크게 흔들렸고, 어떤 구간에서 수익과 손실이 벌어졌는지를 먼저 보는 도구입니다.
같은 단지라도 언제 사느냐에 따라 수익률이 -25%에서 +82%까지 100%p 이상 차이가 날 수 있습니다.
그 차이가 어디서 오는지를 이해하면 현재 시점에 대한 판단 근거가 생깁니다.

수익률 분석을 이해하려면 먼저 **갭투자** — 전세를 끼고 아파트를 매수하여 실제로 투입하는 자기자본을 줄이는 투자 방법 — 의 구조를 알아야 합니다.

예를 들어 매매가 10억, 전세가 6억인 아파트를 매수하면 내 돈(실투자금)은 4억입니다.
2년 후 매매가가 12억이 되면 매매차익은 2억이고, 수익률은 2억 ÷ 4억 = 50%입니다.
아파트 가격이 20% 올랐는데 수익률은 50%가 됩니다.
이것이 **레버리지 효과** — 적은 자본으로 큰 자산에 투자하여 수익률을 극대화하는 구조 — 입니다.

다만, 집값이 떨어지면 손실도 레버리지 만큼 커집니다.

> ❓ **궁금해요 전세가율이란?**
>
> 전세가율은 매매가 대비 전세가의 비율입니다.
> 매매가 5억, 전세가 3억이면 전세가율은 60%입니다.
> 전세가율이 높을수록 실투자금(갭)이 줄어들어 레버리지 효과가 커집니다.
> 단, 전세가율이 70~80%를 넘으면 집값 하락 시 전세금을 돌려주지 못하는 깡통전세 위험이 생깁니다.

실제 수익률은 보유기간과 개인의 세금 부담에 따라 달라질 수 있습니다. 
개별적인 상황을 모두 고려하여 수익률을 계산하는 것은 어렵기 때문에, 부동산 수익률은 세금 관련 부분은 제외하고 보통 **2년 보유**를 기준으로 계산합니다.
양도소득세도 2년 이상 보유하면 혜택이 적용되고, 갭투자 시 세입자의 계약기간이 기본 2년인 경우가 많기 때문입니다.

## 1단계: 분석 프레임워크 설정하기

이 실습은 2단계로 진행됩니다.
1단계에서 AI에게 분석 역할과 규칙을 부여하면, AI는 사용자 입력 폼을 출력합니다.
2단계에서 폼을 채워 제출하면 AI가 실거래 데이터를 수집하고 수익률을 계산합니다.

2단계로 나누는 이유는 정확도 때문입니다.
분석 대상과 기간이 먼저 확정되어야 AI가 불필요한 가정을 하지 않고 데이터 수집에 집중합니다.

### 프롬프트를 보내기 전에

1단계 프롬프트는 분석 역할, 계산 규칙, 출력 형식을 AI에게 설정합니다.
이 단계에서 AI는 분석을 시작하지 않고, 2단계 사용자 입력 폼만 출력합니다.
면적 해석 규칙이 포함되어 있어 "84㎡"와 "34평"을 정확히 구분하여 처리합니다.
이 실습의 핵심도 결국 같습니다. 계산할 수 있는 것은 계산하고, 전세 데이터처럼 부족한 것은 `확인 필요`로 남기는 태도가 더 중요합니다.

### 실행 방법

1. ChatGPT에서 새 대화창(New Chat)을 엽니다.
2. 웹 검색 기능을 켤 수 있으면 켭니다.
3. 아래 1단계 프롬프트를 전체 복사하여 붙여넣습니다.
4. AI가 사용자 입력 폼을 출력하면 1단계가 완료된 것입니다. 아직 아무 숫자도 입력하지 않습니다.

💡 **Tip.** 최신모델을 선택하세요. 이 프롬프트는 구조화된 데이터 처리가 많으므로 최신 모델에서 더 정확하게 작동합니다.

아직 어떤 데이터도 분석하지 않았으므로, 2단계에서 실제 조건을 입력합니다.

### 확인할 포인트

| 확인 항목                       | 기준                                 | 문제 시 대처                                                |
| ------------------------------- | ------------------------------------ | ----------------------------------------------------------- |
| AI가 폼 출력 전에 분석을 시작함 | 입력 폼 먼저 출력해야 정상           | "분석하지 말고 사용자 입력 폼만 먼저 출력해주세요"라고 요청 |
| 폼에 면적 입력 안내가 없음      | 전용㎡/평형 구분 안내 포함 여부 확인 | 프롬프트 전체를 다시 복사하여 새 대화창에 입력              |
| 필수 항목 목록이 다름           | 분석 기간, 단지명, 지역, 면적 4가지  | 프롬프트 [필수 항목] 섹션 확인                              |

## 2단계: 실제 조건 입력하고 분석받기

1단계에서 AI가 출력한 폼에 실제 분석 조건을 채워 제출하면 분석이 시작됩니다.
폼에 조건을 입력할 때는 대괄호(`[ ]`) 안의 내용만 본인 조건으로 교체하면 됩니다.

### 프롬프트를 보내기 전에

2단계 입력 폼은 분석 대상과 기간을 확정하는 역할을 합니다.
단지명, 지역, 면적을 정확히 입력해야 AI가 올바른 실거래 데이터를 찾습니다.
민락푸르지오 84㎡를 2020~2024년으로 분석하는 예시 입력을 참고하세요.

### 실행 방법

1. 1단계에서 AI가 출력한 폼을 확인합니다.
2. 아래 예시 형식을 참고하여 본인 조건에 맞게 폼을 채웁니다.
3. 채운 폼을 같은 대화창에 그대로 붙여넣어 전송합니다.
4. AI가 필수 항목을 모두 확인하면 데이터 수집과 수익률 계산을 시작합니다.

💡 **Tip.** 아실(asil.kr)에서 해당 단지의 연도별 실거래가를 미리 확인해두면 AI 응답 결과를 빠르게 검증할 수 있습니다.

### 프롬프트

```markdown
[사용자 입력 폼]

| 항목 | 입력값 |
|---|---|
| 분석 기간 | [YYYY]년 ~ [YYYY]년 |
| 단지 1(필수) | [단지명] |
| 단지 2(선택) | [단지명 또는 없음] |
| 지역(시/구/동) | [시/구/동] |
| 면적 입력(아래 중 최소 1개는 필수입니다) |  |
| - 전용면적(㎡) | [예: 59 / 84 / 59㎡ / 84㎡ / 모름] |
| - 평형(평, 통상 공급면적 기준) | [예: 24평 / 34평 / 모름] |

### 면적 입력 안내(참고)
- "59/84"처럼 **숫자만** 입력하시면 기본적으로 **전용㎡**로 간주됩니다.
- "24평/34평"처럼 **평**으로 입력하시면 기본적으로 **평형(공급면적 기준)**으로 간주됩니다.
- **평형만 입력**하셨고 전용㎡ 매핑이 불명확하면, 분석 전에 **1~2개 확인 질문**이 있을 수 있습니다.

(선택) 추가 정보
- 내가 아는 전세 정보: [예: 2022.10 전세 2.5억 / 실거래, 없으면 없음]
- 특이사항: [재건축/재개발/역세권/학군/대형공급/산업단지 등]
- 당시 체감: [거래 절벽/회복/과열 등]
```

### AI 응답 확인하기

의정부 민락푸르지오 84㎡를 기준으로 다음과 같은 결과가 나옵니다.

| 매수 구간         | 매도 구간         |    매수가(억) |    전세가(억) |  실투자금(억) |    매도가(억) | 매매가 상승률(%) | 갭투자 수익률(%) | 비고                |
| ----------------- | ----------------- | ------------: | ------------: | ------------: | ------------: | ---------------: | ---------------: | ------------------- |
| 2020.12           | 2022.10           |           4.8 |           2.4 |           2.4 |           4.2 |           -12.5% |           -25.0% | 하락장              |
| 2021.09           | 2023.09           |           5.5 |           2.6 |           2.9 |           5.0 |            -9.1% |           -17.2% | 고점 매수           |
| **2022.10** | **2024.11** | **4.2** | **2.5** | **1.7** | **5.6** | **+33.3%** | **+82.4%** | **저점 매수** |

이 표에서 핵심은 매수 시점에 따른 수익률 차이입니다.
2021년 고점에 매수했으면 -17.2% 손실이고, 2022년 저점에 매수했으면 +82.4% 수익입니다.
같은 단지인데 매수 시점이 1년 다를 뿐인데 수익률이 100%p 차이가 납니다.

2022년이 저점이었던 이유는 금리 급등과 규제 강화로 매수 심리가 얼어붙었기 때문입니다.
모두가 공포에 빠져 매도하던 시기였지만, 돌이켜보면 그때가 최고의 매수 타이밍이었습니다.

전세가율과 실투자금의 관계도 결과에서 확인할 수 있습니다.

| 연도   | 전세가율 | 실투자금 | 레버리지 효과 |
| ------ | -------- | -------- | ------------- |
| 2021년 | 47%      | 2.9억    | 낮음          |
| 2022년 | 60%      | 1.7억    | 높음          |
| 2024년 | 45%      | 3.1억    | 보통          |

2022년은 전세가율 60%로 실투자금이 1.7억에 그쳤습니다.
내 돈을 적게 넣었으니 수익률이 증폭된 것입니다.
반면 2021년은 전세가율 47%로 실투자금이 2.9억이나 필요했습니다.

AI 분석 결과에는 현재 시점과 과거를 비교한 섹션도 포함됩니다.

| 비교 항목 | 과거 강세장 환경(2020) | 현재 환경(2026) | 판정 |
| --------- | ---------------------- | --------------- | ---- |
| 기준금리  | 0.5%                   | 3.25%           | 불리 |
| 전세가율  | 69%                    | 52%             | 불리 |
| 입주물량  | 부족                   | 적정            | 중립 |
| 정책 방향 | 규제 강화              | 규제 완화 중    | 유리 |

2020~2022년 같은 +100% 이상 수익률을 기대하기는 어렵습니다.
현재는 회복 초기 구간으로, 2022년 저점보다는 올랐지만 2021년 고점보다는 낮습니다.
적정가 범위 내 매물이라면 매수할 수 있지만, 과거와 같은 폭발적 상승은 기대하지 않는 것이 현실적입니다.

### 확인할 포인트

| 확인 항목            | 기준                       | 문제 시 대처                                                |
| -------------------- | -------------------------- | ----------------------------------------------------------- |

> ⚠️ **주의:** AI가 출력하는 실거래가, 전세가 수치는 반드시 아실(asil.kr) 또는 국토부 실거래가 공개시스템에서 직접 확인해야 합니다. AI의 분석 프레임워크는 신뢰할 수 있지만, 개별 숫자는 오류 가능성이 있습니다.

## 핵심 정리

- **타이밍이 수익률을 결정한다**: 같은 단지라도 매수 시점에 따라 수익률이 -25%에서 +82%까지 100%p 이상 차이가 납니다.
- **갭투자의 구조**: 전세를 끼고 실투자금을 줄이면 레버리지 효과로 수익률이 증폭되지만, 하락 시 손실도 같은 비율로 커집니다.
- **빠른 정리는 AI, 최종 확인은 공식 사이트**: AI가 제공하는 분석 프레임워크는 활용하되, 실거래가 숫자는 아실과 국토부 사이트에서 반드시 교차검증합니다.
- **데이터가 협상력이다**: 단지 비교, 적정가, 협상 목표가, 과거 수익률로 쌓인 분석 결과는 매수 결정의 근거이자 중개사 협상에서 구체적 숫자로 활용됩니다.
