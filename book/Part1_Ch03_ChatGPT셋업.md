# ChatGPT를 부동산 비서로 만들기

AI 도구를 비교하고 ChatGPT를 메인으로 선택했습니다.
이 챕터에서는 ChatGPT Plus 가입부터 프로젝트 지침 설정, 그리고 첫 프롬프트 실행까지 완료합니다.
챕터가 끝나면 ChatGPT가 내 투자 조건을 기억하는 맞춤형 부동산 비서가 됩니다.

**학습 목표**
- ChatGPT Plus 가입과 GPT-5.4 모델 선택을 완료합니다
- 프로젝트를 만들고 지침에 나의 투자 조건과 답변 방식을 설정합니다
- 첫 프롬프트(구매력 계산)를 실행하여 설정이 제대로 되었는지 확인합니다

**핵심 키워드**
ChatGPT Plus / GPT-5.4 / 프로젝트 / 지침 / 메모리 / 구매력 계산

## 셋업에 필요한 3가지 준비

본격적인 실습에 앞서, AI 투자 비서를 만들기 위한 최소한의 도구를 준비해야 합니다.

## ChatGPT 인터페이스 익히기

모든 준비가 끝났으면 ChatGPT 화면을 살펴보겠습니다. chatgpt.com에 접속하면 다음과 같은 화면이 나옵니다.

```
┌─────────────────────────────────────────────────────────┐
│  ☰ ChatGPT                                    [계정]    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  [+ 새 채팅]                                            │
│                                                         │
│  프로젝트:                                               │
│  - 📁 부동산 투자 분석                                    │
│                                                         │
│  최근 대화 목록:                                         │
│  - 서울 아파트 분석                                      │
│  - 대출 계산 문의                                        │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│             대화 내용 표시 영역                           │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  [📎] [메시지 입력...]                          [전송↗] │
└─────────────────────────────────────────────────────────┘
```

| 버튼 | 위치 | 기능 |
|------|------|------|
| **+ 새 채팅** | 좌측 상단 | 새로운 대화 시작 |
| **☰ 메뉴** | 좌측 상단 | 프로젝트, 대화 기록 접근 |
| **📎 클립** | 입력창 좌측 | 파일 업로드 (엑셀, PDF 분석 가능, 유료) |
| **계정** | 우측 상단 | 설정, 요금제 변경 |

파일 업로드 기능을 활용하면 "이 실거래가 데이터 분석해줘"라고 하면서 엑셀 파일을 올려 ChatGPT가 분석하도록 할 수 있습니다. 이 기능은 유료(Plus) 사용자만 이용할 수 있습니다.

## GPT-5.4 모델 선택하기

ChatGPT에는 여러 모델이 있습니다. 이 책에서 사용하는 **GPT-5.4**가 가장 강력합니다.

| 모델 | 특징 | 사용 조건 |
|------|------|----------|
| **GPT-4o mini** | 기본 모델, 빠르지만 성능 제한 | 무료 |
| **GPT-5.4** | 최신 모델, 빠르고 강력 | Plus 필요 |
| **o3** | 추론 특화 모델, 복잡한 논리에 강함 | Plus 필요 |

대화 창 상단의 모델 선택 드롭다운을 클릭하고 "GPT-5.4"를 선택하면 됩니다.

💡 **Tip.** 무료 버전으로도 기본적인 질문은 가능합니다. 하지만 복잡한 계산이나 파일 분석은 GPT-5.4가 훨씬 정확합니다. 예산이 허락한다면 Plus 구독을 추천합니다.

## 프로젝트를 만들어 부동산 비서를 구성하기

ChatGPT에게 매번 "나는 무주택자이고, 예산은 얼마이고..."라고 설명하는 것은 번거로운 일입니다.

**프로젝트** 기능을 사용하면 이 문제를 해결할 수 있습니다. 프로젝트는 관련 대화를 하나의 폴더로 묶고, **지침**을 설정하여 프로젝트 안의 모든 대화에 자동으로 적용하는 기능입니다. "나는 이런 사람이고, 이런 식으로 답변해줘"라고 한 번만 설정하면, 프로젝트 안에서 새 대화를 열 때마다 자동으로 반영됩니다.

### 프로젝트 만드는 방법

01. 좌측 사이드바에서 [+ 새 프로젝트] 클릭
02. 프로젝트 이름을 "부동산 투자 분석"으로 입력
03. 프로젝트가 생성되면 프로젝트 화면 상단의 [지침 추가] 클릭
04. 지침 입력란에 투자 조건과 답변 방식을 작성
05. 작성 후 자동 저장됨

### 프로젝트 지침이란

프로젝트 지침은 프로젝트 안의 **모든 대화에 자동으로 적용**되는 사전 설정입니다. 일반 채팅의 Custom Instructions와 비슷하지만, 프로젝트 단위로 관리되기 때문에 용도별로 분리할 수 있다는 장점이 있습니다.

| 항목 | 일반 채팅 | 프로젝트 |
|------|----------|----------|
| 지침 적용 범위 | 모든 채팅에 적용 | 해당 프로젝트 채팅에만 적용 |
| 용도별 분리 | 하나만 설정 가능 | 프로젝트마다 다른 지침 설정 가능 |
| 파일 첨부 | 채팅마다 개별 첨부 | 프로젝트에 파일을 올려 전체 채팅에서 참조 |
| 대화 관리 | 시간순 나열 | 프로젝트별 폴더로 정리 |

예를 들어, "부동산 투자 분석" 프로젝트에는 투자 조건 지침을, "일상 대화" 프로젝트에는 다른 지침을 설정할 수 있습니다. 이 책에서는 **"부동산 투자 분석" 프로젝트**를 만들고 그 안에서 모든 실습을 진행합니다.

## 프로젝트에 참고 파일 올리기

프로젝트에는 지침 외에 **파일**을 올려둘 수도 있습니다. 프로젝트 화면에서 [파일 추가] 버튼을 클릭하면 PDF, 엑셀, 텍스트 파일 등을 업로드할 수 있습니다. 업로드한 파일은 프로젝트 안의 모든 대화에서 자동으로 참조됩니다.

| 활용 예시 | 설명 |
|----------|------|
| 실거래가 엑셀 | 관심 단지의 최근 실거래가 데이터를 올려두면 언제든 분석 요청 가능 |
| 대출 조건표 | 은행별 대출 금리·조건을 정리한 파일을 올려 비교 분석 |
| 임장 메모 | 현장 방문 후 작성한 메모를 올려 AI와 함께 검토 |

## 설정 전후, 답변이 이렇게 달라집니다

같은 질문을 해도, 프로젝트 지침 설정 여부에 따라 답변이 완전히 달라집니다.

**설정 전 (일반 채팅):**
```
질문: "내가 살 수 있는 아파트 가격은?"

ChatGPT: 아파트 구매 가능 가격은 개인의 재정 상황, 대출 조건,
지역 등 여러 요소에 따라 달라집니다.
일반적으로 LTV(담보인정비율)는 40~70%이며,
DTI(총부채상환비율)는 40~60%가 적용됩니다.
정확한 계산을 위해 은행 상담을 권장합니다.
```
→ **일반론만 나옵니다.** 구체적 계산이 없습니다.

**설정 후 (프로젝트 안에서 질문):**
```
질문: "내가 살 수 있는 아파트 가격은?"

ChatGPT: 홍길동님의 조건으로 계산해드리겠습니다.

| 항목 | 계산 | 금액 |
|------|------|------|
| 스트레스 DSR 기준 대출 | 연소득 × DSR / (금리+1.5%) | 약 2.6억 |
| 실제 대출 가능액 | LTV·DSR 중 낮은 값 | 약 2.6억 |
| 최대 매수 가능가 | 현금 + 대출 | 약 5.1억 |

추천 예산: 안전 마진 고려 시 4억 5천만 원 이하
```
→ **맞춤형 계산과 구체적 금액이 나옵니다.** 이것이 프로젝트 지침의 힘입니다.

프로젝트 안에서 새 대화를 열 때마다 지침이 자동 적용되므로, 매번 조건을 설명할 필요가 없습니다.

---

## 메모리 기능으로 대화 맥락 유지하기

ChatGPT에는 **메모리** 기능도 있습니다. 프로젝트 지침은 내가 미리 알려주는 정보이고, 메모리는 대화하면서 ChatGPT가 스스로 기억하는 정보입니다.

| 기능 | 특징 | 예시 |
|------|------|------|
| 프로젝트 지침 | 사용자가 직접 입력·수정, 프로젝트 내 전체 적용 | "나는 31세 무주택자" |
| 메모리 | 대화 중 자동 저장, 전체 ChatGPT에 적용 | "지난번에 하남시 관심 있다고 했습니다" |

예를 들어 "지난번에 하남시 분석했었는데, 광명시랑 비교해줘"라고 하면, ChatGPT가 이전 대화 내용을 바탕으로 답변해줍니다.

> ⚠️ **주의: 메모리는 설정 화면에서 관리할 수 없습니다.**
> 메모리에 저장된 내용을 확인하거나 수정하려면 **채팅으로 직접 요청**해야 합니다.

**메모리 확인·수정 방법:**

```
① 확인: "네가 나에 대해 기억하고 있는 내용을 모두 보여줘"
② 수정: "내 관심 지역이 하남시가 아니라 광명시야. 수정해줘"
③ 삭제: "하남시 관심 있다는 거 잊어줘"
④ 추가: "내 투자 예산이 6억으로 늘었어. 기억해줘"
```

이처럼 메모리는 모든 관리를 **대화를 통해** 해야 합니다. 별도의 관리 화면이 없기 때문에, 투자 조건이 바뀌었다면 채팅에서 기존 내용을 확인하고 수정을 요청하세요.

이런 점에서 **프로젝트 지침이 더 편리합니다.** 프로젝트 지침은 언제든 직접 열어서 내용을 확인하고 수정할 수 있기 때문입니다.

## 내 구매력 계산하기

모든 설정이 끝났습니다. 이제 설정이 제대로 되었는지 **첫 프롬프트를 실행**해서 확인해 보겠습니다.

### 프롬프트를 보내기 전에

가장 기본적인 질문은 "내가 얼마짜리 집을 살 수 있는가"입니다. 프로젝트 지침에 이미 투자 조건을 입력했기 때문에, 별도로 조건을 말하지 않아도 ChatGPT가 알아서 적용합니다.

01. **"부동산 투자 분석" 프로젝트** 안에서 새 대화를 엽니다.
02. GPT-5.4 모델이 선택되어 있는지 확인합니다.
03. 아래 프롬프트를 복사해서 붙여넣습니다.

### 프롬프트

> **프롬프트**
>
> 나의 투자 조건으로 구매 가능한 최대 아파트 가격을 계산해줘.
>
> 다음 항목을 포함해서 분석해줘:
> 1. 대출 가능 금액 (LTV, 스트레스 DSR 적용)
> 2. 최대 매수 가능가
> 3. 안전 마진 고려한 추천 예산
> 4. 부대비용 (취득세, 중개수수료 등)
> 5. 주의사항
>
> 2025년 10월 15일 규제 기준으로 계산해줘.

프로젝트 지침을 설정하지 않았다면, 프롬프트 앞에 자신의 조건(연소득, 가용현금, 주택 보유 수, 기존 대출 등)을 추가해야 합니다.

### AI 응답 확인하기

프로젝트 지침에 완성형 예시(가용현금 2.5억, 연소득 6,500만원, 무주택자)를 입력했다면 다음과 유사한 결과가 나옵니다.

```
## 입력 조건 (프로젝트 지침 기준)
- 가용 현금: 2억 5천만 원
- 연소득: 6,500만 원
- 주택 보유: 무주택자
- 기존 대출: 없음

## 대출 가능 금액

### LTV 기준 (담보인정비율)
- 적용 LTV: 40% (2025.10.15 규제)
- LTV 기준 대출 = 매매가 × 0.4

### 스트레스 DSR 기준
- 기준 금리: 4.5% (가정)
- 스트레스 금리: 4.5% + 1.5% = 6.0%
- DSR 한도: 40%
- 연간 원리금 상환 가능액: 6,500만 원 × 40% = 2,600만 원
- 30년 만기 기준 대출 가능액: 약 2억 6천만 원

## 최대 매수 가능가

| 항목 | 금액 |
|------|------|
| 가용 현금 | 2억 5천만 원 |
| 대출 가능액 | 2억 6천만 원 |
| **최대 매수 가능가** | **5억 1천만 원** |

## 추천 예산

| 구분 | 금액 | 비고 |
|------|------|------|
| 최대 가능가 | 5억 1천만 원 | 이론적 한계 |
| 부대비용 | 약 2,500만 원 | 취득세, 중개수수료, 이사 등 |
| 비상금 | 약 3천만 원 | 6개월 생활비 |
| **추천 예산** | **4억 5천만 원 이하** | 안전 마진 포함 |
```

이와 유사한 결과가 나오면 프로젝트 지침 설정이 성공한 것입니다. 결과에서 확인할 포인트는 다음과 같습니다.

| 확인 항목 | 기준 |
|----------|------|
| 내 조건이 반영되었는가 | 가용현금, 연소득, 주택 수가 맞는지 |
| 규제가 적용되었는가 | LTV 40%, 스트레스 DSR이 계산에 포함되었는지 |
| 계산 과정이 보이는가 | 단계별 산출 근거가 명시되었는지 |
| 안전 마진이 있는가 | 부대비용, 비상금을 고려한 추천 예산이 있는지 |

💡 **Tip.** 결과가 예상과 다르면 프로젝트 지침을 다시 확인하세요. 특히 연소득, 가용현금, 기존 대출 항목이 정확한지 점검합니다.
