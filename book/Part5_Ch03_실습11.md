# 실습 ⑪: 구매력 계산 시뮬레이션

실습 ⑩에서 A타입 프로필이 정리되었다면, 이제 질문은 하나로 좁혀집니다.
내 조건으로 실제 얼마짜리 집까지 가능한가?
이번 실습에서는 대출 환경을 먼저 조사합니다.
그다음 A타입 입력값을 넣어 구매력을 계산한 뒤, 마지막에 추천 가격대까지 한 번에 정리합니다.
실습 ⑫~⑭는 여기서 나온 결과 요약을 그대로 이어받습니다.

**학습 목표**
- 현재 대출 환경을 최신 기준으로 조사합니다
- A타입 입력값으로 실제 대출 가능액과 실질 구매력을 계산합니다
- 병목 규제와 추천 가격대를 정리해 다음 실습에 넘깁니다

**핵심 키워드**
LTV / DTI / 스트레스 DSR / 절대한도 / 구매력 / 현금 상한선 / 정책대출

---

## 대출 한도와 실제 구매력은 다릅니다

많은 분이 은행 상담에서 "한도가 5억 나옵니다"라는 말을 들으면, 곧바로 7억대 집을 볼 수 있다고 생각합니다.
하지만 실제 의사결정에서 중요한 값은 **대출 한도**가 아니라 **실질 구매력**입니다.

이 둘이 다른 이유는 간단합니다.
대출은 소득으로 계산되지만, 매수는 결국 **현금 + LTV + 규제지역 조건**까지 함께 맞아야 성립하기 때문입니다.

| 구분 | 무엇을 보는가 | 결과 |
|------|--------------|------|
| 대출 한도 | 소득, DTI, DSR, 절대한도 | 빌릴 수 있는 최대 금액 |
| 구매력 | 대출 한도 + 보유 현금 + LTV | 실제 살 수 있는 집값 |

A타입의 최신 테스트 결과는 아래처럼 정리됩니다.

| 항목 | 결과 |
|------|------|
| 가용 현금 | **2.3억** |
| 합산 연소득 | **1.03억** |
| 대출 가능액 | **정책 4.2억 / 일반 4.9억** |
| 실질 구매력 | **정책 6.5억 / 일반 비규제 7.2억 / 규제지역 약 3.83억** |
| 추천 가격대 | **정책 5.0억~5.4억 / 일반 비규제 6.3억~6.5억** |

같은 A타입이라도 규제지역, 비규제지역, 정책대출 가능 여부에 따라 결과가 크게 갈립니다.
그래서 이번 실습은 한 번에 계산하지 않고, 리서치 -> 계산 -> 종합 3단계로 진행합니다.

---

## 대출 환경을 먼저 조사합니다

### 프롬프트를 보내기 전에

이번 턴의 목적은 계산이 아니라 **기준값 확보**입니다.
금리, LTV, 스트레스 DSR, 절대한도, 정책대출 조건을 먼저 정리해야 뒤 계산이 흔들리지 않습니다.

특히 지금은 수도권 대출 규제가 강하게 작동하는 시기라, 스트레스 DSR 가산금리와 절대한도를 최신 값으로 확인하는 것이 중요합니다.
A타입처럼 무주택·예비신혼 조건이라면 보금자리론 가능 여부도 함께 확인해야 합니다.

### 실행 방법

01. 새 대화창을 엽니다.
02. 아래 프롬프트를 붙여넣습니다.
03. 금리, LTV, DSR, 정책대출 조건이 표로 정리되는지 확인합니다.

### 🤖 프롬프트 (Type A - 직접 실행형)

```markdown
[Role]
당신은 부동산 대출 규제 전문 리서처입니다.

[Mission]
주택 구매력 계산에 필요한 현재 대출 환경을 조사해주세요.
아래 항목을 모두 최신 기준으로 표로 정리해주세요.

1. 현재 주담대 금리
- 시중은행 고정/변동 금리
- 30년 원리금균등상환 기준 금리

2. 대출 규제 현황
- 규제지역/비규제지역 LTV
- DTI
- 수도권/비수도권 스트레스 DSR 가산금리
- 주담대 절대한도

3. 정책대출 종류와 조건
- 보금자리론 일반/생애최초
- 디딤돌 일반/신혼
- 소득 요건, 주택가 제한, LTV, 금리, 한도

4. 계산 기준
- LTV 기반 계산식
- DTI/DSR 기반 계산식
- 연상환계수와 월 상환액 계산 방식

[Rules]
- 모든 수치에 출처와 기준일을 명시하세요
- 현재 시점 기준 최신 정보를 사용하세요
- 스트레스 DSR은 반드시 `%p` 수치로 기재하세요
- 발표되지 않은 규제를 창작하지 마세요
```

### AI 응답 확인하기

A타입 예시 결과에서 핵심적으로 확인된 값은 아래와 같습니다.

| 항목 | 확인된 값 |
|------|----------|
| 기준금리 | **2.50%** |
| 수도권/규제지역 스트레스 DSR | **+3.0%p** |
| 비수도권 스트레스 DSR | **+0.75%p** |
| 규제지역 LTV | **40%** |
| 비규제지역 LTV | **70%** |
| 주담대 절대한도(15억 이하) | **6억** |

이 값들이 뒤 계산의 기준점이 됩니다.
실습 ⑪에서 가장 먼저 확인해야 하는 건 결국 수도권 3.0%p, 비규제 LTV 70%, 절대한도 6억입니다.

### 확인할 포인트

| 확인 항목 | 기준 | 문제 시 조치 |
|----------|------|-------------|
| 스트레스 DSR이 `%p`로 나왔는가 | 수도권 +3.0%p 확인 | "3단계 말고 %p 수치로 다시 알려줘" |
| LTV가 지역별로 분리됐는가 | 규제/비규제 구분 | "규제지역과 비규제지역을 나눠서 다시 정리해줘" |
| 정책대출 조건이 있는가 | 보금자리론/디딤돌 비교 | "정책대출 소득 기준과 주택가 제한도 넣어줘" |

## A타입 기준으로 실제 한도를 계산합니다

### 프롬프트를 보내기 전에

이제부터는 누구나 통하는 계산이 아니라 A타입 입력값을 대입한 계산입니다.
현재 테스트 기준 A타입 입력값은 아래처럼 고정되어 있습니다.

| 항목 | 값 |
|------|------|
| 보유 현금 | 2.3억 |
| 합산 연소득 | 1.03억 |
| 혼인 상태 | 혼인신고 전 |
| 보유 주택 | 0채 (무주택) |
| 목적 | 첫집 마련 |

이번 턴에서 중요한 건 계산 순서입니다.
먼저 소득 기반 최대 대출액을 구하고, 그다음 LTV 역산으로 실질 최대 매물가를 구해야 합니다.
매물가를 먼저 정하면 계산이 그 가격에 끌려가 버립니다.

### 실행 방법

01. 같은 대화창에서 계속 진행합니다.
02. 실습 ⑩ 결과 요약을 붙여넣습니다.
03. 아래 프롬프트를 입력합니다.

### ➡️ 후속 프롬프트 (Type D)

이 후속 프롬프트는 같은 대화창에서 Turn 1 결과를 이어 받은 상태로 실행합니다.

```markdown

혼인상태: 혼인신고 전
보유주택: 0채
투자목적: 첫집 마련

위 대출 환경과 투자 프로필을 바탕으로 아래 순서대로 계산해주세요.

1. 정책대출 해당 여부
2. 소득 기반 최대 대출액 (DTI -> DSR -> 절대한도)
3. LTV 기반 실질 최대 매물가
4. 월 상환액
5. 병목 분석

[Rules]
- 매물가를 먼저 설정하지 마세요
- 소득 기반 한도 -> LTV 역산 순서로 계산하세요
- 혼인 전/후 시나리오가 다르면 구분해 설명하세요
```

### AI 응답 확인하기

A타입 예시 계산 결과는 아래처럼 정리됩니다.

| 항목 | 결과 |
|------|------|
| 정책대출 가능액 | **4.2억** |
| 일반주담대 가능액 | **4.9억** |
| 규제지역 실질 최대 매물가 | **약 3.83억** |
| 비규제지역 실질 최대 매물가 | **약 7.20억** |
| 정책대출 실질 최대 매물가 | **약 6.50억** |
| 핵심 병목 | **정책대출은 상품한도 / 일반은 비규제 DSR, 규제지역은 현금+LTV** |

여기서 눈여겨볼 점은 일반주담대로 4.9억 수준을 빌릴 수 있다는 점과 정책대출로 6.5억 수준까지 접근할 수 있다는 점이 서로 다른 전략이라는 점입니다.
일반주담대는 비규제지역에서 총구매력이 더 커지지만 월 상환 부담이 커지고, 정책대출은 총구매력은 다소 작아도 초기 상환 부담이 훨씬 낮습니다.

### 확인할 포인트

| 확인 항목 | 기준 | 문제 시 조치 |
|----------|------|-------------|
| 계산 순서가 맞는가 | 소득 한도 -> LTV 역산 | "매물가 먼저 두지 말고 다시 계산해줘" |
| 규제/비규제/정책대출이 구분되는가 | 3가지 결과가 따로 나와야 함 | "규제/비규제/정책대출 시나리오를 분리해줘" |
| 병목이 명시되는가 | DSR인지 현금+LTV인지 확인 | "어디서 막히는지 한 줄로 정리해줘" |

## 핵심 정리

- **대출 한도와 구매력은 다릅니다**: 한도가 5억 나와도 LTV와 현금 때문에 실제 살 수 있는 집값은 더 낮아질 수 있습니다.
- **A타입의 전략은 둘로 나뉩니다**: 정책대출은 총구매력 약 6.5억, 일반주담대 비규제는 약 7.2억까지 볼 수 있습니다.
- **선택전략은 정책대출 우선입니다**: 추천가격대는 5.0억~5.4억, 일반주담대는 차선으로 6.3억~6.5억을 봅니다.
- **다음 실습 연결**: 이 결과를 실습 ⑫에 넣어 리스크 시나리오와 대응 전략을 계산합니다.
