# Part 9. 갈아타기 실전

갈아타기에도 여러 유형이 있습니다.
이번 Part에서는 그중 갈아타기(다운사이징) — 서울 대형 아파트를 팔고 경기도 소형으로 옮기면서 잔여 현금으로 월 현금흐름을 만드는 케이스를 다룹니다.
C타입의 실습 사례로 **안산 중흥S-클래스 더퍼스트 59㎡**를 매수 후보 단지로 설정하고, 기존 **서울 노원구 상계주공4단지 84㎡**를 매도 대상으로 진행합니다.
이제 해야 할 일은 분석이 아니라 실행입니다. 매도가를 산정하고, 자금을 정산하고, 적정 매수가를 잡고, 계약하고, 잔금까지 끝내는 것입니다.
이제 남은 질문은 하나입니다.
그래서 이 집을, 얼마에, 어떤 순서로, 어떻게 안전하게 살 것인가?
Part 9에서는 매도가 산정, 자금 정산, 적정 매수가 산정, 계약 리스크 점검, 잔금·등기 일정까지 갈아타기 실행의 전 과정을 다룹니다.

이번 Part의 예시는 **C타입 — 은퇴 준비, 갈아타기, 무대출 전액 현금 매수**를 기준으로 진행합니다.
즉, 레버리지를 크게 쓰는 갈아타기 실전이 아니라 서울 집 매도 -> 소형 주택 매수 -> 잔여 현금 확보 흐름에 맞춘 실행 파트입니다.

**학습 목표**
- 기존 주택의 적정 매도가를 AI로 산정합니다
- 매도·매수 전체 자금 정산표를 작성합니다
- 잔여현금의 투자 운용 계획을 안정형/균형형/적극형 3가지 전략으로 설계합니다
- 적정 매수가를 산정하고 협상 전략을 세웁니다
- 계약 전 리스크와 특약 조항을 점검합니다
- 잔금부터 등기까지의 D-day 실행표를 완성합니다

**핵심 키워드**
적정 매도가 / 매도 전략 / 적정 매수가 / 협상 전략 / 매도-매수 동시진행 / 갈아타기 / 특약 / 잔금일 조율 / 등기 이전 / D-day 플래너

---

## 분석이 끝나도 매수는 아직 시작되지 않았습니다

단지를 비교하고 현장까지 확인하면, 많은 분이 거의 끝났다고 느낍니다.
좋은 단지를 골랐고, 현장도 확인했고, 이제 계약만 하면 된다고 생각합니다.

하지만 실제로는 그때부터가 시작입니다.
갈아타기 실전은 분석 능력과 다른 기술을 요구합니다.

| 분석 단계에서 하는 일 | 실행 단계에서 해야 하는 일 |
|----------------------|--------------------------|
| 좋은 단지 선정 | 기존 집 적정 매도가 산정 |
| 적정가 판단 | 매도인과 가격 협상 |
| 현장 하자 확인 | 계약금·잔금 일정 조율 |
| 최종 매수 판단 | 등기부등본·특약 점검 |
| — | 잔금 송금, 소유권 이전, 전입신고 |

좋은 매물을 찾고도 실제 매수 단계에서 실패하는 이유는 대부분 비슷합니다.

- 기존 집의 적정 매도가를 몰라 자금 계획이 틀어진다
- 적정가를 몰라 비싸게 산다
- 자금 정산을 잘못해 잔금일에 현금이 부족해진다
- 등기부등본 위험 요소를 놓친다
- 특약 없이 계약했다가 일정이 틀어진다
- 매도와 매수 잔금일을 맞추지 못해 이사와 자금 흐름이 꼬인다

Part 9는 바로 이 지점을 다룹니다.
즉, 좋은 물건을 찾는 법이 아니라 찾아놓은 물건을 실제로 안전하게 사는 법을 배우는 파트입니다.

---

## C타입의 갈아타기 실전은 일반적인 첫집 매수와 다릅니다

C타입은 A타입이나 B타입과 출발점이 다릅니다.
핵심은 대출 최대치가 아니라 매도-매수 동시 정산입니다.

| 항목 | A타입 / B타입 중심 질문 | C타입 중심 질문 |
|------|----------------------|----------------|
| 자금 | 대출이 얼마나 나오는가 | 매도 순현금이 얼마 남는가 |
| 매수 방식 | 정책대출 / 일반 주담대 | 무대출 전액 현금 |
| 핵심 리스크 | DSR, 금리, 계약금 부족 | 잔금일 불일치, 일시적 2주택, 생활비 구조 |
| 의사결정 포인트 | 최대 구매력 | 잔여 현금과 월 현금흐름 |

이번 예시에서 현재 확정된 기준 매물은 아래와 같습니다.

| 항목 | 값 |
|------|----|
| 독자 타입 | **C타입 — 은퇴 준비 1주택자** |
| 매수 후보 단지 | **안산 중흥S-클래스 더퍼스트** |
| 위치 | **안산시 단원구 선부동** |
| 면적 | **전용 59㎡** |
| 목표 가격대 | **4억 내외** |
| 핵심 이유 | **고대안산병원 10분, 선부역 6분, 신축 2년** |

"어떻게 더 크게 사는가"가 아니라,
"어떻게 더 작게 옮기면서 안전하게 현금을 남기는가"가 이번 갈아타기의 핵심입니다.

---

## 갈아타기 실전은 5단계로 진행합니다

Part 9의 전체 흐름은 아래와 같습니다.

```
C타입 매수 후보 매물 + 기존 주택 매도 조건
         │
         ↓
┌────────────────────────────────────┐
│ ㉜ 매도가 산정                      │
│ → 내 집 얼마에 팔 수 있나           │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ ㉝ 자금 정산 + 투자 계획             │
│ → 팔면 돈이 얼마 남고, 남은 돈을    │
│   어떻게 굴릴 것인가                 │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ ㉞ 적정 매수가 산정과 협상 전략      │
│ → 그 돈으로 얼마에 사야 하나         │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ ㉟ 계약 전 리스크 점검과 특약        │
│ → 어떤 조건으로 계약하나             │
└────────────────┬───────────────────┘
                 ↓
┌────────────────────────────────────┐
│ ㊱ 잔금·등기 D-day 플래너           │
│ → 언제 어떤 순서로 끝내나            │
└────────────────┬───────────────────┘
                 ↓
         내 이름으로 등기 완료
```

실습별 역할을 정리하면 이렇습니다.

| 실습 | 핵심 질문 | 핵심 산출물 |
|------|----------|------------|
| ㉜ | 내 집을 얼마에 팔 수 있는가 | 적정 매도가 범위 + 매도 전략 |
| ㉝ | 매도와 매수를 합치면 돈이 어떻게 남는가 + 잔여현금 투자 계획 | 매도 정산표 + 매수 정산표 + 잔여 현금 + 안정형/균형형/적극형 운용 계획 |
| ㉞ | 이 매물, 얼마에 사야 적정한가 | 적정가 범위 + 협상 전략 |
| ㉟ | 계약 전에 무엇을 막아야 하는가 | 체크리스트 + 갈아타기 특약 |
| ㊱ | 잔금일부터 등기까지 어떤 순서로 움직여야 하나 | D-day 실행표 + 일정 통합표 |

이번 Part는 각 실습이 서로 강하게 연결됩니다.
㉜의 매도가가 ㉝의 자금 구조를 결정하고, ㉝의 자금이 ㉞의 협상 범위를 결정하고, ㉞의 확정가가 ㉟의 계약 조건을 결정하고, ㉟의 특약이 ㊱의 일정 동기화 기준이 됩니다.
자금 구조를 먼저 파악해야 협상 범위가 보이고, 확정가가 있어야 계약 점검이 가능합니다.

## Part 9를 시작하려면 이것이 필요합니다

Part 9는 C타입의 갈아타기 사례를 직접 설정하고 진행합니다.
아래 정보가 준비되면 바로 시작할 수 있습니다.

| 필요한 정보 | Part 9에서 쓰는 이유 |
|------------|------------------|
| 매수 후보 매물 정보 | ㉞ 적정매수가 산정, 협상 기준 |
| 기존 주택 정보 (단지, 면적, 보유기간) | ㉜ 매도가 산정 |
| 기존 주택 대출 잔액 | ㉜~㉝ 매도 순현금, 자금 구조 |
| 별도 저축/퇴직금 | ㉝ 총 가용현금 계산 |

특히 C타입은 아래 값이 중요합니다.

| 항목 | 현재 C타입 기준 |
|------|---------------|
| 기존 주택 | 서울 1주택 보유 |
| 매수 방식 | 무대출 전액 현금 |
| 핵심 전략 | 선매도 후매수 |
| 핵심 체크포인트 | 매도 잔금일과 매수 잔금일 조율 |

즉, Part 9의 핵심은 단순히 "좋은 계약"이 아니라
매도-매수-이사-등기 전체를 하나의 프로젝트로 관리하는 것입니다.

---

## Part 9를 시작하기 전에 이것만 확인하면 됩니다

아래 다섯 가지가 준비돼 있으면 Part 9를 시작할 수 있습니다.

| 준비물 | 예시 |
|--------|------|
| 매수 후보 매물 정보 | 단지명, 전용면적, 호가, 층/향 |
| 매수 후보 호가 | 협상 목표가, 상한선 |
| 매수 의사결정 | 매수 / 조건부 매수 / 보류 |
| 기존 주택 매도 조건 | 매도 예정가, 대출 잔액, 잔금 예상일 |
| 기존 주택 기본 정보 | 단지명, 전용면적, 층/향, 보유 기간 |

이 값이 없으면 실습 ㉜부터 바로 막힙니다.
특히 C타입은 현재 집을 얼마에 팔고, 얼마가 실제로 남는가가 매수 가능 여부를 좌우하므로, 매도 조건 정리가 필수입니다.

## 핵심 정리

- **Part 9는 실행 파트입니다**: 분석이 아니라 매도가 산정, 자금 정산, 계약, 특약, 잔금·등기처럼 실제 갈아타기 절차를 다룹니다.
- **C타입의 핵심은 무대출 전액 현금 + 매도-매수 동기화입니다**: 대출보다 잔금일 맞추기와 잔여 현금 확보가 더 중요합니다.
- **5개 실습은 순서대로 연결됩니다**: 매도가 산정 -> 자금 정산 -> 적정가 -> 계약 리스크 -> D-day 플래너 흐름입니다.
- **다음 실습 연결**: 실습 ㉜에서 기존 주택의 적정 매도가를 AI로 산정하는 것부터 시작합니다.
