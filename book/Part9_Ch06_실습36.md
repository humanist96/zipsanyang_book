# 실습 ㊱: 매도·매수 통합 D-day 플래너

자금을 정산하고, 적정가를 확정하고, 특약도 준비했습니다.
이제 정말 남은 것은 실행뿐입니다.
이번 실습에서는 매도와 매수를 하나의 일정표로 묶고, 잔금일 맞추기 시나리오와 이사 계획까지 포함한 D-day 플래너를 만듭니다.

**학습 목표**
- 매도와 매수 일정을 하나의 타임라인으로 통합합니다
- 잔금일 조율 시나리오를 정리합니다
- 매도 지연 대비 3단계 시나리오를 확인합니다
- 실제로 들고 움직일 수 있는 D-day 체크리스트를 만듭니다

**핵심 키워드**
D-day 플래너 / 매도-매수 동시진행 / 잔금일 조율 / 근저당 말소 / 등기 이전 / 전입신고 / 이사 1회 / 매도 지연

## C타입의 마지막 리스크는 가격이 아니라 일정입니다

C타입은 무대출 전액 현금 구조이기 때문에, 마지막 병목은 대출 심사보다 일정입니다.
특히 아래 두 날짜가 핵심입니다.

| 항목 | 날짜 |
|------|------|
| 매도 주택 | 서울 노원구 상계주공4단지 84㎡ |
| 매도 잔금일 | **2026년 8월 10일** |
| 매수 잔금일 | **2026년 8월 12일** |

현재 일정은 같은 날 잔금이 아니라, 매도를 먼저 끝내고 1~2영업일 여유를 두는 방식입니다.
이 짧은 여유를 어떻게 흡수할지가 실전의 핵심입니다.

## D-day 체크리스트 만들기

### 프롬프트를 보내기 전에

이번 실습은 매도와 매수를 한 장에 올리는 것이 핵심입니다.
매도 잔금일 오전에 돈을 받고, 그 돈으로 대출을 상환하고, 며칠 안에 매수 잔금을 치르는 흐름이 자연스럽게 연결되어야 합니다.

### 실행 방법

01. 새 대화창을 엽니다.
02. 실습 ㉟ 결과 요약을 붙여넣습니다.
03. 아래 프롬프트를 입력합니다.

이 프롬프트를 실행하면 매도-매수 통합 D-day 역산표와 매도 지연 대비 시나리오를 받을 수 있습니다.

매도와 매수를 통합한 D-day 체크리스트와 잔금일 조율 시나리오를 작성해주세요.

- 매도 지연 시 대응 시나리오(1주 지연 / 1개월+ 지연 / 매도 무산)를 포함하세요
- 법적 기한(전입신고, 취득세 등)은 최신 기준으로 정리하세요

### AI 응답 확인하기

이번 일정안에서 가장 중요한 포인트는 아래 세 가지입니다.

1. **매도 잔금일 2026-08-10**
2. **매수 잔금일 2026-08-12**
3. **기존 주담대 1.8억을 매도 잔금일에 상환**

이번 일정은 같은 날 잔금이 아니라, 2영업일 짧은 공백을 관리하는 방식입니다.

### 잔금일 맞추기 시나리오

| 시나리오 | 의미 | 평가 |
|----------|------|------|
| 같은 날 | 매도 오전, 매수 오후 | 이상적이지만 말소·송금 시간 리스크 큼 |
| 1~2영업일 차 | 단기 공백 관리 | **현재 케이스에 가장 가깝고 현실적** |
| 2주 이상 갭 | 임시 거처 필요 가능성 | 피하는 것이 좋음 |

이번 조건에서는 8월 10일 매도 -> 8월 12일 매수 구조이므로, 핵심은 1~2영업일 동안의 서류 정리·자금 이동·이사 준비를 어떻게 정리할지입니다.
실제 테스트에서도 같은 날 동시잔금보다 매도 선행 후 완충이 더 안전한 구조로 정리됐습니다.

### 통합 D-day 체크리스트

| 시점 | 해야 할 일 |
|------|------------|
| 매도 잔금 6주 전 | 매수자와 잔금 희망일 1차 조율, 기존 주담대 상환 방식 은행 확인 |
| 매도 잔금 4주 전 | 근저당 말소 필요서류·비용 확인, 이사 업체 2~3곳 견적 |
| 매도 잔금 3주 전 | 잔금일 재확정, 관리비·장기수선충당금 정산 기준 확인 |
| 매도 잔금 2주 전 | 말소서류 준비 일정 확정, 소유권이전 등기 일정 법무사와 확정 |
| 매도 잔금 1주 전 | 매수인 잔금 준비 상태 최종 확인, 은행 방문 시간 예약 |
| 매도 잔금 3일 전 | 기존 집 관리비·공과금 미납 여부 점검, 새집 하자 재확인 |
| 매도 잔금일 | 잔금 수령 -> 기존 대출 1.8억 상환 -> 근저당 말소 접수 |
| 매도 후 1영업일 | 서류 정리/계좌 확인/이사 대기 |
| 매수 전일 | 잔금 자금 최종 점검, 새집 상태 재확인 |
| 매수 잔금일 | 잔금 송금 -> 소유권 이전 -> 열쇠 인수 -> 이사 진행 |
| 입주 직후 | 전입신고, 관리사무소 명의 변경 |
| 60일 이내 | 취득세 신고·납부 |

### 당일 타임테이블 감각

**매도 잔금일 (8월 10일)**
- 오전: 매도 잔금 수령
- 오전~정오: 기존 주담대 1.8억 상환
- 오후: 말소 확인, 남은 자금 정리

**매수 잔금일 (8월 12일)**
- 오전: 매수 잔금 송금
- 정오 전후: 등기 서류 확인
- 오후: 열쇠 인수, 현장 점검, 이사 진행

즉 이 실습의 핵심은 한 날에 다 끝내는 기술이 아니라, 짧은 갭을 사고 없이 넘기는 기술입니다.

## 매도 지연 대비 시나리오 3단계

### 1단계: 매도가 1주 지연되면

대응 원칙: **매수 잔금일 연장 요청**

- 실습 ㉟ 특약 중 **매도 연동 / 매도 지연 연장 / 잔금일 조율 특약**을 즉시 활용
- 목표는 **매수 잔금도 같은 폭으로 뒤로 미루는 것**
- 이사 일정도 함께 슬라이드

실무 액션:
1. 매도 잔금 지연 통보를 받은 즉시
2. 같은 날 매수 측 중개사·매도인에게 공식 통지
3. 특약 근거로 **1주 연장** 요청
4. 법무사·이사업체 일정 재조정

이 경우는 **특약 기반 연장으로 충분히 방어 가능**한 수준입니다.

### 2단계: 매도가 1개월 이상 지연되면

대응 원칙: **임시 거처 + 계약 유지 가능성 재판단**

검토 포인트:
- 새집 매도인이 장기 연장을 받아줄지
- 계약금 몰취/해제 리스크가 특약으로 충분히 커버되는지
- 임시거처 비용이 얼마나 드는지

필요 대응:
- 단기 월세 또는 가족집 체류 검토
- 보관이사 비용 반영
- 새집 계약 유지 vs 해제 중 경제성 비교
- 중개사 통해 **잔금 연장 합의서** 별도 작성

이 단계부터는 일정 맞추기가 아니라 손실을 줄이는 쪽으로 넘어갑니다.

### 3단계: 매도 무산 시

대응 원칙: **특약 기반 매수 계약 해제 절차 검토**

핵심:
- 실습 ㉟에서 잡아둔 **매도 연동 특약**이 가장 중요
- 특약 문구가 약하면 계약금 리스크가 생길 수 있음
- 특약이 명확하면 "기존 주택 매도 불성립"을 해제 사유로 주장 가능

실무 액션:
1. 매도 무산 사실 입증자료 확보
2. 매수 측에 즉시 서면 통지
3. 특약 조항 기준으로 해제 또는 연장 협의
4. 계약금 반환 여부를 중개사·법무사 통해 정리

**정리하면**:
- 1주 지연: 연장으로 방어
- 1개월+ 지연: 임시거처 포함 재설계
- 매도 무산: 특약 기반 해제 검토

## 법적 기한과 후속 행정

| 항목 | 기한 |
|------|------|
| 전입신고 | 전입 후 14일 이내 |
| 취득세 신고·납부 | 취득일로부터 60일 이내 |
| 양도세 신고 | 양도일이 속한 달 말일부터 2개월 이내 |

잔금 치르고 나면 다 끝난 것 같지만, 전입신고·취득세·양도세 신고가 남아 있습니다.
갈아타기는 매도와 매수가 동시에 돌아가기 때문에, 잔금 직후 기한을 놓치기 쉽습니다.

### 확인할 포인트

| 확인 항목 | 기준 | 문제 시 대처 |
|----------|------|-------------|
| 매도·매수 일정이 한 표에 정리되는가 | 통합 타임라인 | "매도와 매수를 한 표로 묶어줘" |
| 잔금일 갭 시나리오가 있는가 | 같은 날 / 단기 갭 / 장기 갭 | "갭 시나리오를 나눠서 정리해줘" |
| 매도 지연 시나리오가 있는가 | 1주 / 1개월+ / 무산 | "매도 지연 대비도 넣어줘" |
| 주담대 상환 순서가 있는가 | 매도 잔금일 상환 | "기존 대출 상환 순서도 넣어줘" |
| 전입·취득세 기한이 있는가 | 14일 / 60일 | "후속 행정 기한도 같이 적어줘" |

## 핵심 정리

- **C타입의 마지막 병목은 일정입니다**: 가격과 자금보다 매도-매수 잔금일 맞추기가 더 중요합니다.
- **이번 일정은 1~2영업일 여유를 두는 방식입니다**: 같은 날 잔금이 아니라, 짧은 공백을 안전하게 넘기는 것이 포인트입니다.
- **매도 잔금일에는 기존 대출 상환이 핵심입니다**: 이 흐름이 어긋나면 전체 일정이 꼬입니다.
- **매도 지연은 3단계로 대비합니다**: 1주 지연은 연장, 1개월+ 지연은 재설계, 무산 시 해제 검토입니다.
- **Part 9 마무리 연결**: 이제 적정가, 자금 정산, 계약 리스크, D-day 실행표까지 모두 갖춘 상태입니다.
