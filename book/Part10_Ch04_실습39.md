# 실습 ㊴: 실행 후 행정 처리 (전입신고·취득세·양도세)

입주가 끝났다고 갈아타기가 끝난 것은 아닙니다.
법으로 정해진 세 가지 기한이 기다리고 있습니다.
**전입신고 14일, 취득세 60일, 양도세 2개월**입니다.

이 기한을 놓치면 과태료나 가산세가 붙을 수 있습니다.
금액보다 더 중요한 건, 뒤 일정과 서류 정리가 함께 꼬일 수 있다는 점입니다.
이번 C타입은 **선매도 후매수**라 일시적 2주택 처분기한을 실제 변수로 다루는 케이스는 아닙니다.
대신 **양도세 비과세 요건**과 **법정 신고 기한**을 정확히 챙기는 것이 핵심입니다.

이번 실습의 핵심은 단순합니다.
**입주 완료 이후 챙겨야 할 법정 기한 3종의 캘린더를 만들고, 각 신고의 필요 서류를 미리 정리하는 것**입니다.

**학습 목표**

- 입주 완료 이후 챙겨야 할 전입신고·취득세·양도세 기한을 정리합니다
- 각 신고의 필요 서류 리스트를 확보합니다
- C타입에서 꼭 확인할 세무 포인트를 구분합니다
- 세무사 상담이 필요한 항목을 식별합니다

**핵심 키워드**
전입신고 / 취득세 / 양도세 / 1세대 1주택 비과세 / 가산세 / 세무사 상담

---

## 법정 기한 3종은 시작일이 다릅니다

세 기한의 **기준일**이 각각 다르다는 점이 까다롭습니다.

| 신고 | 기한 | 기준일 | 담당 기관 |
|------|-----|--------|----------|
| 전입신고 | 14일 이내 | **전입일**(실제 거주 시작일) | 주민센터 |
| 취득세 | 60일 이내 | **취득일**(잔금일 또는 등기일 중 빠른 날) | 지방자치단체 |
| 양도세 | 2개월 이내 | **양도일이 속한 달의 말일** | 국세청(홈택스) |

C타입 예시로 날짜를 계산하면 이렇습니다.

```
매도 잔금일: 2026-08-10 → 양도일
매수 잔금일: 2026-08-12 → 취득일
입주일: 2026-08-12 → 전입일

전입신고 마감: 8/12 + 14일 = 2026-08-26
취득세 마감: 8/12 + 60일 = 2026-10-11
양도세 마감: 8월 말일(8/31) + 2개월 = 2026-10-31
```

세 기한이 서로 다른 시점에 몰려 있습니다.
각 기한을 캘린더에 표시해 두지 않으면 놓치기 쉽습니다.

## 프롬프트를 보내기 전에

이번 실습은 독자 본인의 **취득일·양도일·입주일**을 AI에게 입력해 개인화된 캘린더를 받는 것이 목적입니다.

| 항목 | 내 정보 입력란 | 예시 |
|------|---------------|------|
| 매도 잔금일 (양도일) | [ ] | 2026-08-10 |
| 매수 잔금일 (취득일) | [ ] | 2026-08-12 |
| 입주일 (전입일) | [ ] | 2026-08-12 |
| 기존 주택 취득일 | [ ] | 2019년 (보유 7년) |
| 기존 주택 실거주 기간 | [ ] | 5년 |
| 기존 주택 조정대상지역 여부 | [ ] | 취득 당시 확인 필요 |
| 매수 주택 유형 | [ ] | 1세대 1주택 (갈아타기) |

기존 주택의 취득 당시 조정대상지역 여부는 양도세 비과세 요건(2년 거주) 판단의 핵심입니다.
명확하지 않으면 세무사에게 확인해야 합니다.

### 실행 방법

01. 새 대화창을 엽니다.
02. 위 항목을 채웁니다.
03. 아래 프롬프트를 붙여넣습니다.
04. AI가 기한 캘린더·서류 리스트·세무사 상담 항목을 정리해주는지 확인합니다.

이 프롬프트를 실행하면 법정·행정 기한 3종의 개인화된 캘린더와 신고별 서류 리스트를 받을 수 있습니다.

### 🤖 프롬프트

```markdown
[Role]
당신은 한국 주택 매매 후속 절차를 정리하는 세무·행정 체크리스트 코디네이터입니다.
전입신고, 취득세, 양도세의 신고 기한과 준비 서류를 보수적으로 정리하고,
단정이 어려운 항목은 반드시 "세무사 확인 필요" 또는 "관할 기관 확인 필요"로 표시하세요.

[Mission]
아래 조건으로 전입신고·취득세·양도세의 기한 캘린더를 작성하고,
각 절차의 필요 서류, 주의사항, 1세대 1주택 비과세 검토 포인트를 정리해주세요.

세율, 공제, 중과, 감면, 처분기한 관련 구체 수치는 본문에 쓰지 말고

[Context — 거래 정보]

- 거래 유형: 선매도 후매수 (이번 예시는 일시적 2주택 해당 없음)

2. 전입신고는 주민등록 행정 절차, 취득세·양도세는 세무 신고라는 점을 구분해 작성

4. 양도세 비과세 검토 시 아래를 구분해 정리

6. 일시적 2주택은 이번 케이스에 왜 적용되지 않는지 한 줄로 설명하고, 거래 순서가 달라지면 별도 검토 필요라고 표시
7. 취득세는 주택 수·가액·취득 시점에 따라 달라질 수 있으므로 "최신 기준 확인 필요"로 표기
```

## AI 응답 확인하기

AI가 캘린더를 만들면, 결과는 아래와 같이 정리됩니다.

### 법정·행정 기한 3종 캘린더 (예시)

| 절차명 | 기준일 | 계산 방식 | 마감일 | 담당기관 |
|------|--------|----------|--------|----------|
| 전입신고 | 전입일 2026-08-12 | 전입일부터 14일 이내 | **2026-08-26** | 주민센터(온라인 가능) |
| 취득세 | 취득일 2026-08-12 | 취득일부터 60일 이내 | **2026-10-11** | 시·군·구청(위택스) |
| 양도세 | 양도일이 속한 달 말일 2026-08-31 | 그 달 말일부터 2개월 이내 | **2026-10-31** | 국세청(홈택스) |

세 기한이 **8월 말 ~ 10월 말** 구간에 몰려 있습니다.
취득세와 양도세가 20일 차이로 붙어 있어, 한쪽을 놓치면 다른 쪽도 놓치기 쉽습니다.

### 전입신고 서류·절차

**필요 서류**
- 신분증(세대주)
- 전세·매매계약서(간혹 요구)

**절차**
1. 주민센터 방문 또는 정부24 온라인 신청
2. 세대주·세대원 정보 입력
3. 접수 즉시 완료 (영수증 보관)

**주의**: 세대주 변경이나 세대 합가·분가가 있다면 현장 방문이 편리합니다.
신고가 늦어지면 과태료가 발생할 수 있고, 다른 행정 처리도 함께 밀릴 수 있습니다.

### 취득세 신고·납부 서류·절차

**필요 서류**
- 매매계약서 사본
- 취득세 신고서 (위택스 작성 또는 대리인 확인)
- 주민등록등본
- 부동산 거래 신고필증
- 등기 관련 서류 (법무사 또는 지자체 안내에 따라 확인)

**절차**
1. 취득일부터 60일 이내 신고·납부 일정 확인
2. 위택스 또는 시·군·구청에서 신고
3. 직접 신고할지, 법무사와 함께 진행할지 미리 정함
4. 납부 영수증과 신고 내역 보관

**세율 (2026년 기준 — 확인 필요)**: 주택 수·취득가액·시점에 따라 달라집니다.
C타입 케이스(4.10억, 1주택)는 낮은 세율 구간으로 검토할 가능성이 높지만, **실제 적용 세율과 세부 항목은 위택스·지자체 기준으로 확인**해야 합니다.

**지연 시**: 가산세가 붙을 수 있으므로 마감일보다 조금 앞서 처리하는 편이 안전합니다.

### 양도세 신고·납부 서류·절차

**필요 서류**
- 양도소득세 신고서 (홈택스)
- 매매계약서 (매도 원본)
- 취득 당시 매매계약서·등기부등본
- 취득·양도 관련 경비 영수증(중개보수, 취득세, 법무사 비용, 수리비 등)
- 1세대 1주택 확인 서류 (주민등록초본 — 주소이력 포함)

**절차**
1. 양도일이 속한 달의 말일부터 2개월 이내
2. 홈택스 양도소득세 신고
3. 납부 세액 계산 (비과세 시 0원, 과세 시 세무사 상담 권장)
4. 신고서 제출 + 납부

**비과세 판정**: 1세대 1주택과 2년 이상 보유를 기본으로 보고, 취득 시점 기준 조정대상지역 여부에 따라 거주 요건을 추가 검토합니다.
C타입 예시는 7년 보유·5년 실거주이므로 요건 충족 가능성이 높지만, **취득 당시 조정대상지역 여부**를 세무사에게 반드시 확인해야 합니다.

### 1세대 1주택 비과세 요건 체크 (C타입 예시)

| 요건 | 기준 | C타입 예시 | 충족 |
|------|------|----------|------|
| 1세대 1주택 | 세대 전체 1주택 | 매도 시점 1주택 | ✅ |
| 보유 기간 | 2년 이상 | 7년 | ✅ |
| 거주 기간 | 조정대상지역 취득 주택이면 2년 이상 거주 검토 | 5년 | 적용 시 충족 가능 |
| 고가주택 여부 | 양도 당시 고가주택 기준 초과 여부 확인 | 8.35억 | 기준상 초과 가능성 낮음 |

C타입 예시는 비과세 요건 충족 가능성이 높습니다.
단, 취득 당시 조정대상지역 여부·세대 구성 변동은 세무사 확인 필수입니다.

### 이번 케이스의 추가 확인 포인트

1. **일시적 2주택 해당 없음**: 이번 C타입은 **선매도(8/10) 후매수(8/12)** 구조라, 새 집을 살 때 이미 기존 주택을 처분한 상태입니다.
2. **조정대상지역 여부 확인**: 기존 주택 취득 당시 지역 기준에 따라 거주 요건 판단이 달라질 수 있습니다.
3. **세대 구성 변동 확인**: 양도일 현재 1세대 1주택 판정에 영향을 주는 변동이 있었는지 점검해야 합니다.

다만 독자 자신의 거래 순서가 `선매수 후매도`라면 이야기가 달라집니다.
그 경우에는 일시적 2주택 규정을 별도로 확인해야 하므로, 이 실습의 달력과 별개로 세무사 상담을 먼저 받는 편이 안전합니다.

### 세무사 상담이 필요한 질문 5가지

1. 기존 주택 취득 당시(2019년) 조정대상지역 여부 확인이 가능한가요?
2. 1세대 1주택 비과세 요건이 최종 충족됐나요? 누락된 서류는 없나요?
3. 양도세 신고 시 꼭 챙겨야 할 필요경비와 증빙은 무엇인가요?
4. 취득세 신고·납부를 직접 할지, 법무사와 함께 진행할지 어떤 방식이 더 적절한가요?
5. 이 거래에서 놓치기 쉬운 신고나 서류가 더 있나요?

본문에 적힌 기한과 세율은 2026년 4월 기준의 개념적 설명이며, 실제 신고 시점에는 반드시 아래를 확인해야 합니다.

- **국세청 홈택스** (양도세·종합부동산세)
- **위택스** (취득세·지방세)
- **세무사 상담** (비과세 판정, 필요 서류, 예외 규정 확인)

이 실습의 산출물은 **기한 캘린더와 서류 리스트**입니다.
초보자라면 세액 계산과 양도세 신고는 세무사와 함께 진행하는 편이 안전합니다.

## 핵심 정리

- **법정 기한은 기준일이 각각 다릅니다**: 전입일·취득일·양도일 말일이 각기 다른 시점에서 카운트됩니다.
- **C타입 기한은 8월 말 ~ 10월 말에 몰립니다**: 전입 8/26, 취득세 10/11, 양도세 10/31 순서입니다.
- **취득세는 직접 신고하거나 법무사와 함께 진행할 수 있습니다**: 방식은 달라도 마감일과 최종 책임은 본인에게 있습니다.
- **양도세 비과세 판정은 세무사 확인이 원칙**: 보유·거주 요건과 조정대상지역 여부를 반드시 전문가에게 검증받습니다.
- **세율·기한은 매년 변경됩니다**: 이 실습은 기한 관리와 서류 준비 틀이고, 세액 계산은 신고 시점에 세무사와 함께합니다.
- **다음 실습에서는 남은 현금을 실제로 굴리는 단계로 넘어갑니다**: 분할 매수·상품 배분·월 현금흐름 실행입니다.
