# 실습 ⑯: 교통·인프라 비교

실습 ⑮에서 후보 지역 5곳이 정해졌다면, 이제 가장 먼저 확인할 것은 교통과 생활 인프라입니다.
이 두 요소는 실거주 만족도와 환금성을 동시에 좌우합니다.
A타입처럼 첫집 마련 목적이라면, '학군'만큼이나 '출퇴근 동선'과 '생활 편의시설'이 중요합니다.

**학습 목표**
- 후보 지역 5곳의 교통 접근성과 생활 인프라를 비교합니다
- 각 지역의 상대적 강점과 약점을 점수로 확인합니다
- 다음 실습의 학군 분석과 합쳐서 실거주 관점의 우선순위를 좁힙니다

**핵심 키워드**
교통 접근성 / 생활 인프라 / GTX / 환승 부담 / 실거주 만족도 / 후보 지역 비교

---

## 교통과 인프라는 실거주 체감 차이를 가장 먼저 만듭니다

같은 59㎡ 아파트라도 어디에 있느냐에 따라 매일의 체감은 크게 달라집니다.
강남 접근 시간이 35분인지 70분인지, 대형마트와 병원이 생활권 안에 있는지에 따라 실거주 만족도가 완전히 바뀝니다.

이번 실습에서는 실시간 길찾기처럼 절대값을 맞추는 것이 목적이 아닙니다.
후보 지역 간 **상대 비교**가 핵심입니다.

---

## 교통·인프라 비교 분석하기

### 프롬프트를 보내기 전에

이번 비교는 실습 ⑮에서 확정한 5개 후보 지역을 그대로 사용합니다.
같은 대화창에서 이어가는 이유는, 후보 지역을 다시 입력하지 않고 바로 비교할 수 있기 때문입니다.
대화가 길어져 응답 품질이 흔들리면, 직전 결과 요약만 들고 새 대화창으로 옮겨 이어갑니다.

### 실행 방법

01. 실습 ⑮와 같은 대화창에서 계속 진행합니다.
02. 아래 프롬프트를 붙여넣습니다.

실행하면 초등·중등·고등·학원가 점수와 종합 순위가 함께 정리됩니다.

### ➡️ 후속 프롬프트 (Type D)

```markdown
위 후보 지역 5곳을 기준으로 교통과 생활 인프라를 비교해주세요.

1. 강남 / 여의도 / 판교 접근 시간
2. 현재 운행 노선과 예정 노선
3. 대형마트 / 종합병원 / 백화점 / 공원 등 생활 인프라
4. 교통 점수 + 인프라 점수 + 종합 순위
```

### AI 응답 확인하기

이번 비교 예시에서 교통과 인프라는 아래처럼 정리되었습니다.

**교통 점수**

| 순위 | 지역 | 점수 |
|------|------|------|
| 1위 | **용인시 수지구 풍덕천동 / 수지구청역 생활권** | **9.3/10** |
| 2위 | **의왕시 내손동 / 인덕원 생활권** | **8.7/10** |
| 3위 | **안양시 만안구 석수동 / 석수역 생활권** | **8.3/10** |
| 4위 | **군포시 산본동 / 산본역·금정역 생활권** | **7.6/10** |
| 5위 | **수원시 영통구 영통동·망포동 생활권** | **7.1/10** |

**생활 인프라 점수**

| 순위 | 지역 | 점수 |
|------|------|------|
| 1위 | **수원시 영통구 영통동·망포동 생활권** | **8.8/10** |
| 2위 | **용인시 수지구 풍덕천동 / 수지구청역 생활권** | **8.6/10** |
| 3위 | **군포시 산본동 / 산본역·금정역 생활권** | **8.2/10** |
| 4위 | **의왕시 내손동 / 인덕원 생활권** | **7.8/10** |
| 5위 | **안양시 만안구 석수동 / 석수역 생활권** | **7.6/10** |

핵심 해석은 단순합니다.

| 지역 | 한 줄 해석 |
|------|-----------|
| 수지구청역권 | **교통 최강 벤치마크** |
| 영통·망포 | **인프라 최강 실거주형** |
| 의왕 내손동 | **교통·호재·예산 균형형** |
| 석수동 | **서울 접근 우수형** |
| 산본동 | **생활 인프라 안정형** |

실거주 관점에서 보면 '의왕 내손동 vs 영통·망포 vs 석수동'의 3파전으로 좁혀지는 이유가 여기서 드러납니다.
수지는 가장 빠른 교통 벤치마크이지만 가격 부담이 크고, 영통·망포는 생활 인프라가 가장 강하며, 의왕은 교통호재와 예산 적합성이 함께 작동합니다.
즉 이 실습은 단순 순위표가 아니라, **어떤 생활권이 어떤 방식으로 유리한지 성격을 분리해 주는 단계**입니다.

### 확인할 포인트

| 확인 항목 | 기준 | 문제 시 대처 |
|----------|------|-------------|
| 교통과 인프라가 분리돼 있는가 | 점수 2세트 | "교통과 인프라 점수를 따로 정리해줘" |
| 강남/판교 접근이 비교되는가 | 상대 비교 가능 | "강남과 판교 기준으로 다시 비교해줘" |
| 예정 노선이 구분되는가 | 현재 vs 예정 분리 | "현재 노선과 예정 노선을 나눠서 써줘" |

💾 **인계 정보** — 교통점수 순위, 인프라점수 순위, 교통·인프라 종합 해석

## 핵심 정리

- **교통 최강은 수지구청역권입니다**: 신분당선 직결성과 강남 접근성에서 벤치마크 역할을 합니다.
- **인프라 최강은 영통·망포입니다**: 대형 상권, 병원, 생활 편의시설이 실거주 만족도를 끌어올립니다.
- **의왕 내손동은 균형형 실매수 후보입니다**: 교통호재와 예산 적합성을 함께 가져가는 지역입니다.
