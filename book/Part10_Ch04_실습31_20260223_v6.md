# 실습 ㉛: 안 팔릴 때 매도 전략 다시 잡기

매물을 내놓고 한 달, 두 달이 지나도 집을 보러 오는 사람이 없거나, 보고 간 사람들에게 아무 피드백이 없을 때 매도자의 마음은 조급해지기 시작합니다.
이 실습에서는 안 팔리는 진짜 원인을 5가지 축으로 진단하고, 가격 인하 외의 대안을 포함한 14일 재가동 실행 플랜을 AI에게 도출합니다.

**학습 목표**

- 매도 지연의 원인을 5가지 축(가격, 상품성, 노출, 유통, 수요 환경)으로 구조적으로 진단합니다
- 가격 인하 외의 대안(상품성 개선, 노출 전략, 중개사 재접근)을 포함한 실행 플랜을 수립합니다
- 최악의 경우를 대비한 플랜B(조건 완화→전략 전환→매도 보류)를 사전에 설계합니다

**핵심 키워드**
매도 지연 / 5축 진단 / 14일 재가동 플랜 / 상품성 개선 / 중개사 피드백 루프 / 플랜B(보류·전환)

---

## 근거 없는 희망 회로와 패닉 셀, 둘 다 위험하다

매물이 장기간 안 팔릴 때 초보 매도자가 빠지는 함정은 두 가지입니다.
"내 집이 좋은데 언젠간 팔리겠지"라며 근거 없이 기다리거나, 반대로 극도의 불안감에 수천만 원씩 호가를 낮추는 **패닉 셀**<sup>공포에 의한 급매</sup>을 하는 것입니다.

안 팔리는 데는 반드시 이유가 있습니다.
가격이 문제인지, 집의 상태(상품성)가 문제인지, 아니면 중개사의 브리핑 우선순위에서 밀린 것(노출/유통)인지 정확한 원인 진단이 선행되어야 합니다.

이 실습의 프롬프트는 매도자의 감정을 배제하고 현재 상황을 5가지 축으로 냉정하게 진단합니다.
무턱대고 3천만 원을 내리는 것보다, 도배 지원금 300만 원을 조건으로 내거는 것이 훨씬 적은 비용으로 매수자의 관심을 끌 수 있습니다.
AI는 가격 인하 외의 다양한 대안을 찾아주고, 도저히 팔 수 없는 최악의 경우에는 반전세·월세 전환 같은 엑시트 출구 전략까지 함께 제시합니다.

---

## 1단계 공용 프롬프트로 진단 환경 설정하기

실습 ㉙~㉚과 동일한 2단계 프롬프트 구조를 사용합니다.
1단계에서 AI에게 미매각 진단 컨설턴트 역할을 부여하고, 2단계에서 현재 매도 상황과 반응 데이터를 입력합니다.

### 프롬프트를 보내기 전에

이 프롬프트의 진단 프레임은 **5가지 축**으로 구성됩니다.

| 축            | 진단 내용                                   |
| ------------- | ------------------------------------------- |
| 가격/조건     | 호가가 시세 대비 적정한가, 네고 조건이 경쟁력 있는가 |
| 상품성        | 내부 상태, 수리 이력, 옵션이 매수자 기대에 부합하는가 |
| 노출          | 매물 사진·설명이 충분한가, 조회수·저장수가 정상인가 |
| 유통(중개 운영) | 중개사 수, 피드백 루프, 브리핑 우선순위가 확보되었는가 |
| 수요 환경     | 해당 지역·타입의 매수 수요가 충분한가          |

5축 중 어디가 병목인지에 따라 처방이 완전히 달라지므로, 2단계 폼에서 "최근 2~4주 반응" 데이터를 가능한 한 구체적으로 입력하는 것이 중요합니다.

### 실행 방법

1. ChatGPT에서 새 대화창(New Chat)을 엽니다.
2. 최신모델을 선택하고, 웹 검색 기능을 켭니다.
3. 아래 1단계 프롬프트 전체를 복사해서 붙여넣습니다.
4. AI가 확인 메시지를 출력하면 2단계로 넘어갑니다.

💡 **Tip.** 네이버부동산이나 직방 앱에서 내 매물의 조회수·저장수를 미리 확인해 두면 2단계 입력이 수월합니다. 중개사에게 "최근 문의가 몇 건이었는지" 물어보는 것도 좋습니다.

### 프롬프트

```
# [안 팔릴 때 쓰는 매도 전략 '다시 잡기' | STEP 1]

[ROLE]
당신은 "미매각/매도 정체 진단 컨설턴트 + 실행 플래너"입니다.
목표는 원인을 구조적으로 진단하고 가격·조건·노출·유통 전략을 재설계하는 것입니다.

[진단 프레임 5축]
1) 가격/조건
2) 상품성
3) 노출
4) 유통(중개 운영)
5) 수요 환경

[입력 검증 게이트]
- 필수 데이터 누락 시 분석 금지.

[OUTPUT FORMAT]
1) 현 상황 요약
2) 5축 진단 + 유력 원인 TOP3
3) 14일 재가동 실행 플랜
4) 플랜B 3단계(완화→전환→보류)
5) 스크립트
6) 추가 질문
```

### AI 응답 확인하기

1단계 프롬프트를 보내면 AI는 STEP 2 입력 안내를 출력합니다.
입력 검증 게이트가 작동하므로 2단계 폼을 제출하기 전까지 진단을 시작하지 않습니다.

### 확인할 포인트

| 확인 항목                        | 기준                           | 문제 시 대처                            |
| -------------------------------- | ------------------------------ | --------------------------------------- |
| AI가 STEP 2 안내를 출력했는가    | 본인 거주/세입자 거주 선택 안내 | 1단계 프롬프트 전체가 복사되었는지 확인 |
| 5축 진단 프레임을 인지했는가     | 확인 메시지에 관련 언급        | 없어도 무방, 2단계 결과에서 확인 가능   |

---

## 2단계 입력 폼으로 현재 매도 상황 제출하기

1단계 안내를 확인했다면, 내 상황에 맞는 2단계 폼을 채워 제출합니다.
본인 거주 중이면 STEP 2-A를, 세입자가 거주 중이면 STEP 2-B를 선택합니다.

### 프롬프트를 보내기 전에

2단계 폼에서 가장 중요한 영역은 **"최근 2~4주 반응"**입니다.
조회수, 문의 수, 방문(쇼잉) 수, 네고 요구 내용은 AI가 병목을 진단하는 핵심 데이터입니다.
정확한 숫자가 없으면 "대략" 또는 "모름"으로 입력해도 되지만, 구체적일수록 진단 정확도가 올라갑니다.

"안 팔리는 이유(본인 추정)" 항목도 중요합니다.
매도자 본인이 느끼는 감(感)과 AI의 객관적 진단을 비교하면, 감정에 가려졌던 진짜 원인이 드러나는 경우가 많습니다.

### 실행 방법

1. 1단계 프롬프트를 보낸 같은 대화창에서 계속 진행합니다.
2. 아래 STEP 2-A 또는 STEP 2-B 중 내 상황에 맞는 폼을 선택합니다.
3. 대괄호 안의 내용을 내 조건으로 바꿔 붙여넣고 전송합니다.
4. 필수 항목이 모두 충족되면 AI가 6개 섹션으로 진단 결과를 출력합니다.

### 프롬프트 — STEP 2-A: 본인 거주용

```
# [매도 전략 '다시 잡기' | STEP2-A 본인 거주용]

※ [] 안의 문장은 입력 가이드입니다. 삭제하지 않아도 됩니다.

[필수]

- 분석 기준일(YYYY-MM-DD):
  [입력 가이드: 오늘 날짜
   예: 2026-03-15]

- 지역/단지명:
  [입력 가이드: 시·구 + 단지명 정도
   예: 수원 영통구 ○○아파트]

- 면적/타입:
  [입력 가이드: 평 또는 전용면적 + 타입
   예: 34평 / 84타입]

- 매물 등록 시작일(대략):
  [입력 가이드: 언제부터 '정식 노출'이 시작됐는지
   예: 2025-11-20]

- 현재 호가(또는 범위):
  [입력 가이드: 현재 광고 가격
   예: 9.2억
   예: 9.2~9.4억]

- 인도 가능 시점:
  [입력 가이드: 실거주라면 인도 타이밍이 수요에 영향
   예: 잔금일 즉시 가능
   예: 잔금 후 1개월 유예 필요]

[최근 2~4주 반응]

- 조회/저장:
  [입력 가이드: 앱/플랫폼 기준, 대략이라도
   예: 조회 1,200 / 저장 35
   예: 모름(중개사에게 확인 필요)]

- 문의 수:
  [입력 가이드: 전화/문자/앱문의 합계
   예: 12건(최근 한 달)]

- 방문(쇼잉) 수:
  [입력 가이드: 실제 집을 본 팀 수
   예: 3팀]

- 네고 요구:
  [입력 가이드: "요구 내용 + 빈도"
   예: 대부분 3천~5천 인하 요구
   예: 인도 유예(1~2개월) 조건 요구 많음]

[운영]

- 중개사 수:
  [입력 가이드: 내놓은 중개사 개수
   예: 2곳 / 5곳]

- 정기 피드백 여부(예/아니오):
  [입력 가이드: 주간 리포트 받는지
   예: 아니오(연락 오면 대응만)]

- 사진/설명 수준:
  [입력 가이드: 콘텐츠 품질
   예: 중개사 촬영, 사진은 보통/설명은 짧음
   예: 사진 어둡고 설명 빈약]

[본인 판단]

- 안 팔리는 이유(본인 추정 1~3개):
  [입력 가이드: 느낌이어도 괜찮음
   예: 가격이 경쟁매물보다 비쌈 / 집이 정리 안 됨 / 역세권이 아님]

- 목표 우선순위(가격/속도/리스크/스트레스):
  [입력 가이드: 하나 또는 순서
   예: 속도 > 리스크 > 가격]

- 피하고 싶은 조건:
  [입력 가이드: 절대 수용하기 싫은 조건
   예: 가격 1억 이상 급락 / 인도 2개월 이상 지연]
```

- 분석 기준일(YYYY-MM-DD):
  [입력 가이드: 오늘 날짜

### AI 응답 확인하기

폼을 제출하면 AI가 6개 섹션으로 매도 정체 진단과 재가동 플랜을 출력합니다.
가장 먼저 확인해야 할 항목은 다음 3가지입니다.

| 순서 | 확인 항목             | 위치   | 왜 중요한가                                           |
| ---- | --------------------- | ------ | ----------------------------------------------------- |
| 1    | 5축 진단 + 유력 원인  | 섹션 2 | 안 팔리는 진짜 원인이 가격인지, 상품성인지, 노출인지 확인 |
| 2    | 14일 재가동 실행 플랜 | 섹션 3 | 오늘부터 2주간 구체적으로 무엇을 할지 확인             |
| 3    | 플랜B 3단계           | 섹션 4 | 14일 후에도 안 팔릴 때 조건 완화→전환→보류 순서 확인  |

AI의 진단이 뼈아프더라도 객관적인 시선으로 받아들이는 것이 중요합니다.
"소장님, 집이 안 나가서 가격 대신 수리비 지원 조건으로 매물 설명을 바꿔서 다시 브리핑해 주실 수 있을까요?"와 같이 적극적으로 행동해야 매도 기간을 단축할 수 있습니다.

> ⚠️ **주의:** AI가 "호가 조정 필요"라고 진단하더라도 무턱대고 수천만 원을 내리지 않습니다. AI의 14일 재가동 플랜을 먼저 실행해 보고, 반응 변화를 확인한 뒤 단계적으로 조정합니다.

### 확인할 포인트

| 확인 항목                           | 기준                                               | 문제 시 대처                                               |
| ----------------------------------- | -------------------------------------------------- | ---------------------------------------------------------- |
| 6개 섹션이 모두 출력되었는가        | 현 상황 요약부터 추가 질문까지 6개                 | 1단계 프롬프트의 OUTPUT FORMAT을 재확인                    |
| 5축 진단이 모두 포함되었는가        | 가격/상품성/노출/유통/수요 환경 5가지              | 누락된 축이 있으면 "5축 진단을 모두 포함해주세요" 요청     |
| 14일 플랜이 날짜별로 구체적인가     | 일별 또는 주별 실행 항목과 담당자가 명시           | "14일 플랜을 날짜별 구체적 행동으로 작성해주세요" 요청     |
| 플랜B가 3단계로 구분되었는가        | 완화(조건 조정)→전환(전략 변경)→보류(매도 중단)   | "플랜B를 3단계로 구분해주세요" 요청                        |

## 핵심 정리

- **5축 진단**: 가격, 상품성, 노출, 유통, 수요 환경 중 어디가 병목인지 파악해야 정확한 처방이 가능합니다.
- **14일 재가동 플랜**: 무작정 가격을 내리기 전에, 사진 교체·중개사 재접근·조건 변경 등 비용이 적은 대안부터 실행합니다.
- **플랜B 3단계**: 14일 후에도 반응이 없으면 조건 완화 → 전략 전환(승계·월세) → 매도 보류 순서로 대응합니다.
- **중개사 피드백 루프**: 진단과 재가동 플랜의 효과를 측정하려면 중개사에게 주간 리포트(문의 수, 네고 요구, 방문 반응)를 요청하는 구조가 필수입니다.
