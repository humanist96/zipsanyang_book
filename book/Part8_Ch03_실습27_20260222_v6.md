# 실습 ㉗: 현장 정보 기반 매물 재평가

실습 ㉖에서 만든 체크리스트를 들고 현장 임장을 다녀왔습니다.
온라인에서 분석했던 내용과 현장에서 직접 확인한 내용이 다른 부분이 있을 것입니다.
이 실습에서는 현장에서 수집한 정보를 AI에게 전달하고, 기존 분석 결과를 수정하여 매물의 등급을 재평가합니다.

**학습 목표**

- 온라인 분석 결과와 현장 확인 결과를 Before vs After로 비교합니다
- 현장 우선의 법칙을 적용하여 데이터 충돌 시 현장 정보를 기준으로 수정합니다
- 중개사 인터뷰로 확인한 실제 거래 가능 가격을 반영하여 재무적 타당성을 재산정합니다

**핵심 키워드**
현장 우선의 법칙 / Before vs After / 가격 현실화 / S/A/B/C 등급 / 재무적 타당성 / Action Plan

---

## 데이터가 충돌하면 현장이 사실이다

임장을 다녀오면 온라인에서 봤을 때와 다른 점이 발견되기 마련입니다.
네이버 지도에서는 역까지 도보 10분이라고 나왔는데 실제로 걸어보니 신호 대기와 경사 때문에 15분이 걸린다거나, "조용한 주거지역"이라는 설명과 달리 실제로는 도로 소음이 있었다거나 하는 식입니다.
반대로 재건축 추진이 생각보다 잘 진행되고 있다는 긍정적인 발견을 하는 경우도 있습니다.
중개사로부터 "이 단지 요즘 문의가 부쩍 늘었어요"라는 말을 들었다면, 이는 수요가 살아나고 있다는 현장 신호입니다.
온라인 데이터에는 아직 반영되지 않은 정보이므로, 이런 내용도 빠짐없이 기록해 두어야 합니다.

**현장 우선의 법칙** — 온라인 데이터와 현장에서 확인한 정보가 충돌하면, 현장 정보를 사실로 간주하여 분석을 수정하는 원칙입니다.

| 항목         | 온라인 데이터       | 현장 확인              | 적용              |
| ------------ | ------------------- | ---------------------- | ----------------- |
| 역까지 거리  | 도보 10분           | 도보 15분 (신호+경사)  | **15분으로 수정** |
| 소음 수준    | "조용한 주거지역"   | 도로 소음 약간 있음    | **소음 있음으로 수정** |
| 호가         | 5.8억               | 실제 협상가 5.5억      | **5.5억으로 수정** |
| 전세 수요    | 불명확              | 1개월 내 계약 가능     | **강함으로 수정** |

이렇게 수정된 데이터를 AI에게 전달하면, AI가 기존 분석과 비교하여 등급을 재산정합니다.

## 등급 판정 기준을 미리 이해하기

AI가 재평가 결과를 S/A/B/C 등급으로 분류합니다. 각 등급의 의미를 미리 알아두면 결과를 해석하기 수월합니다.

| 등급  | 의미                          | 행동                     |
| ----- | ----------------------------- | ------------------------ |
| **S** | 조건 완벽, 즉시 매수 가능     | 가계약 즉시 진행         |
| **A** | 조건 양호, 매수 적합          | 조건 확인 후 매수 진행   |
| **B** | 일부 리스크 있으나 수용 가능  | 추가 확인 후 결정        |
| **C** | 리스크 과다, 매수 부적합      | 보류 또는 다른 매물 탐색 |

등급은 현장 점수표 5개 항목(교통, 쾌적성, 생활 인프라, 가격 매력, 리스크)의 평균과 치명적 결격 사유 유무로 결정됩니다.
1개 항목이라도 1점(매우 나쁨)이면 등급이 한 단계 하향됩니다.
예를 들어 교통 5점, 쾌적성 4점, 생활 인프라 4점, 가격 매력 4점으로 평균이 높더라도, 리스크 항목이 1점이면 A가 아니라 B로 하향됩니다.

현장에서 작성한 체크리스트 전체를 입력해야 정확한 재평가가 가능합니다. 일부만 입력하면 현장의 부정적 정보가 누락되어 등급이 비정상적으로 높게 나올 수 있습니다.

04. AI가 Before vs After 비교표, 상세 재평가, 현장 인사이트, 행동 가이드를 순서대로 출력합니다.

# Critical Rules (핵심 규칙)

## 현장 우선 규칙

1. **Fact Check 필수**: 사전 분석 데이터와 현장 데이터가 충돌할 경우, 무조건 현장 체크리스트의 내용을 사실로 간주하여 분석을 수정 (예: 지도상 10분이나 체감 15분이면 '15분'으로 수정)
2. **변동 방향 표기**: Before → After 변동을 반드시 상향/하향/유지로 명시

## 가격 규칙

3. **가격 현실화**: 중개사 인터뷰로 확인된 '실제 거래 가능 가격(네고 후)'을 기준으로 예산 대비 매수 가능 여부 재판정
4. **부대비용 포함**: 매수가 외에 취득세(~1.1%), 중개수수료(~0.4%)를 합산한 '실제 필요 자금' 산출

## 등급 규칙

5. **S/A/B/C 등급**: 현장 점수 5개 항목 평균 + 치명적 결격 사유 유무로 판정
6. **결격 사유**: 1개 항목이라도 1점(매우 나쁨)이면 등급 한 단계 하향
7. **리스크 재산정**: 현장 반응(주민/중개사 의견)을 반영하여 리스크 가중치 조절

### AI 응답 확인하기

의정부 민락푸르지오 84㎡ 임장 후 재평가를 요청한 경우, 다음과 같은 Before vs After 비교표가 출력됩니다.

| 항목           | AI 초기 진단 (Before) | 현장 재평가 (After) | 변동 |
| -------------- | --------------------- | ------------------- | ---- |
| 역까지 거리    | 도보 10분             | **도보 15분**       | 하향 |
| 소음 수준      | 조용                  | **약간 있음**       | 하향 |
| 실거래 가능가  | 5.5~5.6억             | **5.5억 확인**      | 유지 |
| 전세 수요      | 불명확                | **강함**            | 상향 |
| 관리 상태      | 미확인                | **우수**            | 상향 |
| **종합 등급**  | A-                    | **A**               | 상향 |

이 사례에서 역까지 거리가 늘어났음에도 종합 등급이 오히려 올라간 이유는, 전세 수요가 강하고 관리 상태가 좋고 가격 협상이 가능하다는 것이 확인되었기 때문입니다.
전체적으로 봤을 때 플러스 요인이 마이너스 요인보다 많았던 것입니다.

재무적 타당성 섹션에서는 부대비용까지 포함한 실제 필요 자금이 산출됩니다.
매수가 5.5억에 취득세 약 600만 원, 중개수수료 약 220만 원을 합하면 실제 필요 자금은 약 5.6억입니다.
예산 6억 기준으로 0.4억의 여유가 있으므로 예산 내 진입이 가능하다는 판정이 나옵니다.
만약 이 계산에서 예산을 초과한다면 AI는 "예산 초과 — 가격 재협상 또는 다른 매물 검토 필요"라는 판정을 내립니다.
부대비용은 매수가의 약 1.5%로, 금액이 클수록 무시할 수 없는 규모가 됩니다.

💡 **Tip.** 재무적 타당성 섹션에서 AI가 취득세율이나 중개수수료율을 잘못 적용하는 경우가 있습니다. AI가 산출한 부대비용은 반드시 직접 계산하여 교차 확인하는 것이 안전합니다.

행동 가이드에서는 Best Option(예: "5.5억 이하 매물 확보 시 가계약 진행")과 Plan B(예: "5.8억 고수 매물만 있을 경우 2~3주 대기 후 재협상")가 구체적으로 제시됩니다.

### 확인할 포인트

| 확인 항목                    | 기준                               | 문제 시 대처                            |
| ---------------------------- | ---------------------------------- | --------------------------------------- |
| 현장 정보가 정확히 반영됨    | 15분, 소음, 5.5억 등 모두 반영     | 체크리스트 전체를 빠짐없이 복사         |
| 예산 계산이 정확함           | 부대비용 포함 실제 필요 자금 < 예산 | 취득세·중개수수료를 직접 계산하여 확인  |
| 등급 판정 근거가 합리적      | 상향/하향 이유가 명확              | 변동 방향 표기가 빠져있으면 재요청      |
| 트리거 조건이 현실적         | 해당 가격 매물이 실제 존재하는지   | 네이버 부동산에서 매물 존재 여부 확인   |

> ⚠️ **주의:** AI가 "매수 적합"이라고 판정했더라도, 예산 계산(부대비용 포함)은 반드시 직접 확인해야 합니다. AI는 개인의 정확한 세율이나 대출 조건을 알 수 없습니다.

## 재평가 결론을 한 장으로 정리하는 방법

AI 분석과 검증을 마치면 다음과 같은 형식으로 결론을 정리합니다.

| 항목                | 결과                                |
| ------------------- | ----------------------------------- |
| **타겟 단지**       | 민락푸르지오 84㎡                   |
| **Before 등급**     | A-                                  |
| **After 등급**      | A                                   |
| **핵심 변동**       | 전세 수요 강함 확인 (불명확 → 강함) |
| **최종 판단**       | 조건부 매수 (5.5억 이하 시)         |
| **다음 단계**       | 실습 ㉘ 최종 의사결정 보고서        |

이 결과가 실습 ㉘에서 최종 의사결정 보고서를 작성할 때 입력값으로 사용됩니다.
Before vs After의 핵심 변동이 무엇이었는지를 한 줄로 정리해 두면 실습 ㉘에서 입력할 때 편리합니다.
위 사례에서는 "전세 수요 불명확 → 강함 확인, 등급 A- → A 상향"이 핵심 변동입니다.

## 핵심 정리

- **현장 우선의 법칙**: 온라인 데이터와 현장 정보가 다르면 현장이 사실입니다. 지도상 10분이어도 실제 15분이면 15분으로 수정합니다.
- **가격 현실화**: 중개사가 확인한 실제 거래 가능 가격과 부대비용을 합산하여 예산 내 진입 가능 여부를 재판정합니다.
- **현장 인사이트**: "요즘 문의 많아졌다" 같은 중개사의 한마디는 시장 온도를 알려주는 신호입니다. 온라인 데이터만으로는 알 수 없는 정보입니다.
- **재평가 결과 활용**: Before vs After 등급과 핵심 변동 사항을 정리한 뒤, 실습 ㉘에서 최종 의사결정의 입력값으로 사용합니다.
