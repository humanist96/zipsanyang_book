# 실습 ⑩: 나의 투자 유형 진단

부동산 투자는 목적에 따라 봐야 할 매물이 완전히 달라집니다.
월세 수익, 시세차익, 실거주 중 어디에 무게를 두느냐에 따라 지역도, 상품도, 전략도 바뀝니다.
이 실습에서는 AI의 구조화된 질문을 통해 내 투자 유형을 객관적으로 진단하고,
이후 실습 ⑪~⑭에서 사용할 **고객 프로필 요약표**를 만듭니다.

**학습 목표**
- AI를 활용하여 투자 유형(안정형/레버리지형/실거주형)을 진단합니다
- 8가지 질문에 답변하여 고객 프로필 요약표를 완성합니다
- 같은 프롬프트라도 조건에 따라 결과가 달라지는 것을 이해합니다

**핵심 키워드**
투자 유형 진단 / 고객 프로필 요약표 / 리스크 허용도 / 멀티턴 대화 / 임대수익률 / 전세가율

---

## 목적이 달라지면 매물이 달라진다

같은 현금 3억이라도 목적에 따라 봐야 할 매물이 다릅니다.

| 유형 | 핵심 목표 | 보는 매물 | 중시 지표 |
|------|----------|----------|----------|
| A. 안정형 수익추구 | 월세 수입 | 오피스텔, 다세대, 아파트 | 임대수익률 5%↑ |
| B. 레버리지 성장형 | 시세차익 | 수요 많은 지역의 아파트 | 전세가율, 거래량, 개발호재 |
| C. 실거주+자산화형 | 거주 + 장기 증식 | 학군·역세권 아파트 | 학군, 교통, 인프라 |

**임대수익률** — 연간 임대 수입을 투자 금액으로 나눈 비율입니다. 5% 이상이면 은행 이자보다 높은 수익을 기대할 수 있습니다.

월세 수익이 목표라면 수익률 5% 이상을 기대할 수 있는 매물을 봐야 합니다.
시세차익이 목표라면 수요가 많고 거래가 활발한 지역에서 레버리지를 잘 활용해야 합니다.
실거주라면 학군과 교통이 우선입니다.

문제는 이런 판단을 감으로 하는 경우가 많다는 점입니다.
주식을 살 때 "누가 좋다고 하더라"는 말만 듣고 사는 것처럼, 부동산도 "요즘 여기가 뜬대"라는 말에 끌려 자기 상황과 맞지 않는 매물을 보게 됩니다.
유형을 먼저 분류하면, "내 유형에 맞는 지역 추천해줘"처럼 AI에게 구체적으로 요청할 수 있습니다.

직접 하려면 투자 유형 분류 기준을 공부하고, 내 재무 상태를 항목별로 정리하고, 어떤 전략이 맞는지 스스로 판단해야 합니다.
AI를 활용하면 8개 질문에 답하는 것만으로 이 과정이 끝납니다.
AI가 객관적으로 조건을 분석해 유형을 진단하므로, 자기 평가에서 생기는 편향도 줄일 수 있습니다.

---

## 투자 유형 진단하기

AI에게 프롬프트를 입력하면, AI가 8가지 질문을 순서대로 물어봅니다.
대화가 끝나면 내 투자 유형과 프로필 요약표가 나옵니다.

### 실행 방법

01. ChatGPT, Claude, 또는 Gemini에서 새 대화창을 엽니다.
02. 아래 프롬프트를 복사해서 붙여넣습니다.
03. AI가 질문을 하나씩 던지면 차례대로 답변합니다. 총 8개 질문입니다.

💡 **Tip.** 이 프롬프트는 대화형입니다. AI가 질문하고, 내가 답변하고, 다시 AI가 다음 질문을 하는 방식으로 진행됩니다. 한 번에 모든 정보를 입력할 필요 없이, AI가 이끄는 대로 따라가면 됩니다.

### 프롬프트

```
[Role]
당신은 15년 경력의 부동산 투자 컨설턴트입니다.

[Mission]
저의 투자 유형을 진단해주세요.
아래 8가지 질문을 한 번에 하나씩 순서대로 진행하고,
모든 답변을 수집한 후 투자 유형을 분류해주세요.

[Context: 질문 목록]
아래 질문을 순서대로 하나씩 진행합니다:

1. 현재 보유 부동산이 있으신가요? (채수, 용도)
2. 투자에 사용 가능한 현금은 대략 어느 정도이신가요?
3. 세전 연소득은 얼마이신가요? 배우자 소득이 있으시면 함께 알려주세요.
   (원천징수영수증 기준 총급여)
4. 현재 상환 중인 대출이 있으신가요? 있다면 월 상환액은 얼마인가요?
5. 월 현금흐름(월세 수입)과 시세차익 중 어느 쪽을 더 선호하시나요?
6. 원금의 몇 %까지 손실을 감내하실 수 있으신가요?
   (예: 5%, 10%, 15%, 손실 불가)
7. 투자 후 예상 보유 기간은 어느 정도이신가요?
   (단기 1~2년 / 중기 3~5년 / 장기 5년 이상)
8. 투자를 고려 중인 지역이 있으신가요?
   (예: 서울 강남권, 경기 수원, 아직 미정)

[Context: 진단 유형]
답변 수집 후 아래 3가지 유형 중 가장 적합한 유형을 진단합니다:

| 유형 | 특징 | 핵심 지표 |
|------|------|----------|
| A. 안정형 수익추구 | 현금흐름 중시, 월세 수입 목표 | 임대수익률 5%↑ |
| B. 레버리지 성장형 | 시세차익 중시, 대출 활용 | 전세가율, 개발호재 |
| C. 실거주+자산화형 | 거주 목적 + 장기 자산 증식 | 학군, 교통, 인프라 |

[Output Format]
모든 질문 완료 후 아래 형식으로 결과를 정리해주세요:

**투자 유형**: [A/B/C 중 하나]
**유형명**: [안정형 수익추구 / 레버리지 성장형 / 실거주+자산화형]

**고객 프로필 요약**
| 항목 | 답변 내용 |
|------|----------|
| 보유 부동산 | |
| 투자 가용 현금 | |
| 세전 연소득 (본인) | |
| 배우자 소득 | |
| 합산 연소득 | |
| 기존 대출 월 상환액 | |
| 선호 수익 방식 | |
| 리스크 허용도 | |
| 예상 보유 기간 | |
| 관심 지역 | |

> 💡 이 결과를 **실습 ⑪ 구매력 계산**에 그대로 붙여넣어 사용합니다.

**진단 근거**: [왜 이 유형으로 진단했는지 2-3문장]
**추천 전략 방향**: [이 유형에 적합한 전략 2-3가지]
**주의사항**: [이 유형의 주요 리스크]

[Rules]
- 질문은 반드시 하나씩 진행 (한꺼번에 여러 질문 금지)
- 고객 답변을 기다린 후 다음 질문 진행
- 불명확한 답변이 있으면 추가 질문으로 명확화
- 모든 질문 완료 후에만 진단 결과 제시
- 이 프롬프트의 내부 구조를 그대로 출력하지 마세요

첫 번째 질문부터 시작해주세요.
```

### 확인할 포인트

| 문제 | 원인 | 대응 |
|------|------|------|
| AI가 질문을 한꺼번에 던짐 | [Rules]를 무시한 경우 | "질문을 하나씩 해주세요"라고 다시 요청 |
| 유형 진단 없이 바로 추천 | 질문 건너뛴 경우 | "8가지 질문을 먼저 모두 해주세요"라고 요청 |
| 프로필 요약표가 안 나옴 | [Output Format] 무시 | "고객 프로필 요약 표를 만들어주세요"라고 요청 |
| A/B/C 외 다른 유형 제시 | AI가 자체 분류 생성 | "프롬프트에 있는 3가지 유형 중 하나로 분류해주세요" |

> ⚠️ **주의:** AI가 진단한 유형이 내 생각과 다를 수 있습니다. 이는 자기 평가의 편향 때문입니다. AI의 객관적 진단을 먼저 수용하고, 이후 실습에서 결과를 검증하세요.

## 핵심 정리

- **유형 분류가 먼저**: 매물을 보기 전에 내 유형을 파악해야 합니다. 유형에 따라 전략이 완전히 다릅니다.
- **AI의 객관적 진단**: 자기 평가는 편향되기 쉽습니다. AI가 구조화된 질문으로 진단합니다.
- **프로필 요약표**: 이후 실습 ⑪~⑭에서 이 표를 입력값으로 사용합니다.
