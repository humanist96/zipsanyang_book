# 실습 ㉟: 계약 전 리스크 점검과 특약 생성

자금을 정산하고, 적정 매수가도 확정했습니다.
이제 계약서에 도장을 찍기 전에 마지막으로 확인해야 할 것이 있습니다.
바로 권리관계와 특약입니다.
이번 실습에서는 등기부등본을 기준으로 위험 요소를 점검하고, C타입의 갈아타기 상황에 맞는 특약 조항을 준비합니다.
다만 이 책에서는 실제 개인정보가 담긴 등기부를 사용할 수 없어서, 등기부 원문 없이 진행합니다. 독자 여러분은 실제 계약 때 반드시 최신 등기부를 발급받아 확인하세요.

갑구는 소유권과 소유자 변동을 적는 부분입니다.
을구는 근저당권, 전세권 같은 권리관계를 적는 부분입니다.

**학습 목표**
- 등기부등본의 갑구와 을구를 점검합니다
- 매도·매수 동시진행에서 빠뜨리기 쉬운 위험 요소를 정리합니다
- 갈아타기 상황에 맞는 특약 조항을 준비합니다

**핵심 키워드**
등기부등본 / 갑구 / 을구 / 근저당 / 일시적 2주택 / 특약 / 잔금일 조정 / 계약 해제

---

## 등기부등본은 계약 전 마지막 안전장치입니다

부동산 계약은 가격만 맞는다고 끝나지 않습니다.
매도인이 진짜 소유자인지, 대출이 얼마나 잡혀 있는지, 가압류나 임차권 문제가 없는지까지 확인해야 비로소 계약서를 쓸 수 있습니다.

이번 실습은 특히 C타입에게 더 중요합니다.
이유는 단순합니다.
기존 집 매도와 새 집 매수가 동시에 얽혀 있기 때문입니다.

| 일반 매수 리스크 | C타입 갈아타기에서 더 중요해지는 리스크 |
|----------------|----------------------------------|
| 근저당 말소 | 매도 잔금과 매수 잔금이 안 맞는 문제 |
| 하자 분쟁 | 이사 2번, 임시 거처, 자금 공백 |
| 대출 미승인 | 일시적 2주택 처분기한과 특약 문제 |

C타입의 계약 특약은 단순한 보호 문구가 아닙니다. 매도 잔금과 매수 잔금 사이에서 돈이 안전하게 흘러가도록 잡아주는 역할을 합니다.

## 등기부등본 리스크 점검하기

### 프롬프트를 보내기 전에

이번 실습은 등기부등본 원문이 있을 때 가장 정확합니다.
원문이 없더라도 일반 체크리스트와 특약 초안은 만들 수 있지만, 실제 계약 직전에는 반드시 최신본을 발급받아 다시 확인해야 합니다.

현재 C타입 기준 매수 조건은 아래와 같습니다.

| 항목 | 값 |
|------|----|
| 매수 대상 | 안산 중흥S-클래스 더퍼스트 59㎡ |
| 매수가 | **4.10억** |
| 계약금 | 약 4,100만 원 (10%) |
| 매수 방식 | 무대출 전액 현금 |
| 기존 주택 | 서울 노원구 상계주공4단지 84㎡ 매도 진행 중 |
| 핵심 리스크 | **8/10 매도 -> 8/12 매수** 일정 동기화 |

### 실행 방법

01. 새 대화창을 엽니다.
02. 등기부등본 원문이 있으면 갑구/을구 내용을 붙여넣습니다.
03. 없으면 등기부 미발급 상태로 먼저 체크리스트와 특약 초안을 받습니다.
04. 아래 프롬프트를 입력합니다.

이 프롬프트를 실행하면 등기부등본 체크리스트와 갈아타기 특화 특약 6종을 받을 수 있습니다.

[Context]
- 매수 대상: 안산시 단원구 선부동 중흥S-클래스 더퍼스트, 중층 호실

- 대출 예정: 없음 (전액 현금)
- 입주 계획: 기존 주택 매도 후 즉시 입주

2. 위험도는 높음 / 주의 / 낮음 으로 표시하세요
3. 갈아타기 특화 특약을 우선 제안하세요
4. 법률 확정 판단이 필요한 부분은 전문가 상담 필요로 명시하세요

### 갈아타기 특화 특약 6종

이번 C타입 기준으로는 아래 특약이 가장 중요합니다.

1. **기존 주택 매도 연동 특약**
   - 예시 계약 문구: `매수인의 기존 주택 매도거래가 잔금일 이전 최종 무산된 경우, 매수인은 이를 증빙하여 본 계약의 해제 또는 잔금일 변경을 요청할 수 있고, 구체적 처리 방식은 상호 협의한다.`
   - 메모: 이 조항은 해제 조건과 위약 책임을 어떻게 둘지에 따라 분쟁 포인트가 생길 수 있어서 **법무사 또는 변호사 상담이 필요합니다**.
2. **매도 지연 시 잔금일 연장 특약**
   - 예시 계약 문구: `매수인의 기존 주택 매도 잔금 일정이 불가피하게 지연되는 경우, 본 계약의 잔금일은 매도 잔금일에 맞추어 최대 14일 범위 내에서 연장할 수 있다.`
   - 메모: 연장 가능 기간, 통지 시한, 지연배상 책임을 함께 적는 편이 좋습니다.
3. **일시적 2주택 처분기한 특약**
   - 예시 계약 문구: `매수인은 기존 주택 처분을 전제로 본 주택을 매수하며, 관련 법령상 요구되는 처분기한 준수를 위해 필요한 범위에서 잔금 및 인도 일정을 협의할 수 있다.`
   - 메모: 세법 해석이 개별 사안에 따라 달라질 수 있으니 **법무사 또는 세무사 상담이 필요합니다**.
4. **잔금일 조정 특약**
   - 예시 계약 문구: `매수인의 기존 주택 매도 잔금일과 본건 매수 잔금일은 상호 협의하여 동기화하며, 필요 시 전후 1~2영업일 범위 내에서 조정할 수 있다.`
   - 메모: 이번 건은 이 특약이 가장 중요합니다. 현재 대출이 없고 전액 현금이지만, 실제로는 **기존 주택 매도대금 유입 시점**이 자금 실행 시점입니다.
5. **하자 발견 시 보수 비용 분담 특약**
   - 예시 계약 문구: `계약일 이후 잔금일 전까지 통상적 확인으로 발견하기 어려운 중대한 하자가 확인될 경우, 매도인은 이를 보수하거나 그 비용 상당액을 잔금에서 공제하는 방식으로 정산한다.`
   - 메모: '중대한 하자'의 범위를 예시로 적어두면 더 좋습니다. 예: 누수, 보일러 작동불능, 전기 누전, 배수 불량.
6. **잔금일 등기 이전 지연 시 처리 특약**
   - 예시 계약 문구: `매도인은 잔금일에 소유권이전등기 및 기존 권리 말소에 필요한 서류를 완비하여 제공하고, 서류 미비로 이전이 지연될 경우 그 책임은 매도인에게 있다.`
   - 메모: 판례상 잔금 지급과 소유권이전등기는 동시이행 관계가 원칙이므로, 서류 완비 여부를 분명히 해두는 게 중요합니다.

다만 실제 실습에서 우선순위가 가장 높았던 것은 아래 3개입니다.

- **기존 주택 매도 연동 특약**
- **잔금일 1~2영업일 조정 특약**
- **등기 이전 지연 시 처리 특약**

실무에서는 아래처럼 이해하면 됩니다.

| 특약 | 왜 필요한가 |
|------|-------------|
| 매도 연동 | 기존 집이 안 팔리면 자금 흐름이 꼬이기 때문 |
| 매도 지연 연장 | 기존 집 잔금이 밀리면 매수 잔금도 밀어야 하기 때문 |
| 처분기한 | 일시적 2주택 세금 리스크 때문 |
| 잔금일 조정 | 이사 1회로 끝내기 위해 |
| 하자 분담 | 신축이라도 하자 분쟁이 생길 수 있기 때문 |
| 등기 지연 | 소유권 이전과 말소가 밀릴 수 있기 때문 |

### 확인할 포인트

| 확인 항목 | 기준 | 문제 시 대처 |
|----------|------|-------------|
| 등기부 최신본이 있는가 | 계약 직전 발급본 | "계약 전 최신본 기준으로 다시 점검해줘" |
| 갑구/을구가 나뉘어 정리되는가 | 소유권 / 권리관계 분리 | "갑구와 을구를 나눠서 다시 써줘" |
| C타입 특약이 따로 제시되는가 | 매도 연동, 잔금일 조정 포함 | "갈아타기 특약 중심으로 다시 정리해줘" |
| 법률 면책이 있는가 | 전문가 상담 필요 명시 | "전문가 상담 필요 문구도 넣어줘" |
