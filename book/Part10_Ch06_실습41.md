# 실습 ㊶: 갈아타기 후 90일 생활 점검

지금까지의 실습을 이어보면 이렇습니다.
실습 ㊲에서 잔금일 시퀀스를 집행했고, ㊳에서 이사를 끝냈고, ㊴에서 행정 기한을 정리했고, ㊵에서 생활비 버킷을 분리한 뒤 12주 분할 매수를 시작했습니다.

이제 남은 일은 **설계대로 실제 생활이 굴러가는지 확인하는 것**입니다.
월 154만원 목표 현금흐름의 첫 그림이 보이는지, 생활비 버킷이 계획보다 빨리 줄지 않는지, 관리비와 단지 생활은 예상과 얼마나 다른지를 30일·60일·90일 세 시점에서 점검합니다.

이번 실습의 핵심은 단순합니다.
**세 번의 점검 사이클을 미리 달력에 박고, 각 시점의 자가진단 질문을 준비하는 것**입니다.

**학습 목표**

- D+30일·D+60일·D+90일 세 시점의 점검 항목을 분리합니다
- 예상 월 현금흐름과 실제 수령·지급 일정의 차이를 기록합니다
- 포트폴리오 재조정과 생활비 버킷 점검 기준을 세웁니다
- 갈아타기 성공 여부의 판정 기준을 확인합니다

**핵심 키워드**
30일 점검 / 60일 점검 / 90일 점검 / 월 현금흐름 실현 / 생활비 버킷 / 포트폴리오 재조정 / 생활비 실측 / 자가진단 / 갈아타기 완료 판정

---

## 갈아타기는 점검으로 끝납니다

갈아타기가 성공했는지는 **입주 직후에는 알 수 없습니다**.
월 154만원 설계가 실제로 통장 흐름으로 이어지는지, 안산에서의 생활이 실제로 서울보다 현금흐름에 유리한지는 **최소 3개월은 지나야 첫 그림이 보입니다**.

다만 이번 예시는 ㊵에서 12주 분할 매수를 2026-11-09까지 진행합니다.
그래서 D+90은 장기 성과를 확정하는 시점이라기보다, **매수 완료 직후의 첫 종합 점검과 1차 리밸런싱 판단 시점**에 가깝습니다.

이 3개월 동안 점검해야 할 것이 세 층위로 나뉩니다.

| 층위 | 점검 대상 | 점검 시점 |
|------|----------|---------|
| 거래 | 행정 기한 준수, 정산 분쟁 여부 | D+30 |
| 자금 | 분할 매수 진행률, 첫 현금 유입, 생활비 실측 | D+60 |
| 생활 | 매수 완료, 1차 리밸런싱, 생활 적합성 | D+90 |

각 시점마다 점검의 목적과 질문이 다릅니다.

## AI 응답 확인하기

AI가 캘린더를 만들면, 세 시점의 점검표가 아래와 같이 나옵니다.

### D+30 점검표 — 거래·행정·정산·분쟁 마무리 (2026-09-11 경)

**점검 목적**: 거래·행정·정산이 모두 깨끗하게 마무리됐는가

**자가진단 질문 5개**

1. 전입신고는 기한 내(8/26)에 완료했는가?
2. 매도자·매수자와의 관리비·공과금·장기수선충당금 정산은 마무리됐는가?
3. 입주 당일 발견한 이상은 기록·공유했고 후속 협의가 진행 중인가?
4. 기존 서울 집 매도 건에서 사후 분쟁 신호는 없는가?
5. 취득세 신고 준비(서류·세무사 연락)는 진행 중인가?

**차이 대응**

| 상황 | 대응 |
|------|-----|
| 전입신고 지연 | 즉시 주민센터 방문, 과태료 수용 |
| 정산 분쟁 발생 | 중개사 입회 하 재정산, 기록 문서화 |
| 이상 공유 누락 | 사진·문자 기록 정리 후 즉시 서면 공유 |
| 매도 사후 분쟁 | 매매계약서·특약 재확인, 필요 시 변호사 상담 |

### D+60 점검표 — 생활비·현금흐름·분할 매수 진행 점검 (2026-10-11 경)

**점검 목적**: 분할 매수가 계획대로 진행 중인가 + 첫 현금 유입 일정이 열리기 시작했는가 + 취득세 마감

**자가진단 질문 5개**

1. 생활비 버킷 0.84억은 별도 계좌에서 유지되고 있는가?
2. 12주 분할 매수는 계획한 주차대로 진행 중인가?
3. 첫 이자·분배금 입금 또는 지급 예정 일정이 확인되는가?
4. 안산에서 첫 1~2개월 생활비와 관리비는 계획과 얼마나 차이나는가?
5. 취득세는 10/11 마감까지 납부했는가?

**차이 대응**

| 상황 | 대응 |
|------|-----|
| 분할 매수 일정 지연 | 다음 2주 일정 재조정, 지연 사유 기록 |
| 첫 현금 유입 미확인 | 상품별 지급월·매수 시점 재확인, 조기 단정 대신 기록 비교 |
| 생활비 +20% 초과 | 항목별 차이 원인 분석, 다음 달 지출 조정 |
| 취득세 미납 | 즉시 납부, 가산세 수용 |

### D+90 점검표 — 매수 완료 직후 1차 리밸런싱 검토 (2026-11-10 경)

**점검 목적**: 12주 분할 매수 완료 확인 + 1차 리밸런싱 판단 + 생활 적합성 점검

**자가진단 질문 5개**

1. 분할 매수 12주가 완료됐는가? (11/9 기준)
2. 생활비 버킷·예금·채권 ETF·고배당 ETF 배분이 계획 범위 안에 있는가?
3. 최근 1~2개월 현금 유입 흐름이 목표 154만원과 크게 어긋나지 않는가?
4. 생활비 버킷의 소진 속도가 예상보다 빠르지 않은가?
5. 안산 생활(교통·편의·이웃)에 심리적 적응이 이뤄졌고, 다음 분기 점검 계획이 수립됐는가?

**차이 대응**

| 상황 | 대응 |
|------|-----|
| 배분 비중 ±5%p 이내 | 정상, 1차 리밸런싱은 기록 점검 중심으로 진행 |
| 배분 비중 ±5%p 초과 | 일부 비중 조정 검토, 원인 기록 |
| 현금흐름 미달 지속 | 지급 일정·매수 완료 시점·세전/세후 차이부터 재점검 |
| 생활비 버킷 소진 과다 | 지출 구조 재점검, 현금흐름 목표와 버킷 계획 재설계 |

## 3시점 통합 캘린더

```
2026-08-12  입주 (D-day)
            │
            ├─ 08/26  전입신고 마감
            │
2026-09-11  D+30 1차 점검
            - 거래·행정·정산 마무리
            │
2026-10-11  D+60 2차 점검
            - 취득세 마감일 겸
            - 분할 매수 진행률·첫 현금 유입 일정 확인
            │
2026-10-31  양도세 마감
            │
2026-11-09  분할 매수 12주 완료
            │
2026-11-10  D+90 3차 점검
            - 1차 리밸런싱 점검

            │

```

## 포트폴리오 재조정 의사결정 프레임

생활비, 현금흐름, 자산 배분은 서로 다른 기준으로 봐야 합니다.
생활비와 현금흐름은 **계획 대비 차이율**, 자산 배분은 **목표 비중 대비 ±5%p 이탈 여부**로 판단하는 편이 실무적입니다.

| 항목 | 기준 | 정상 | 주의 | 경계/재설계 |
|------|------|------|------|-------------|
| 월 생활비 | 계획 대비 차이율 | ±10% 이내 유지 | ±20% 이내 원인 분석 | ±30% 수준이면 지출 구조 재설계 검토 |
| 월 현금흐름 | 목표 대비 차이율 | ±10% 이내 유지 | ±20% 이내 지급 일정·세전/세후 차이 재점검 | ±30% 수준이면 운용 전략 재검토 |
| 자산 배분 | 목표 비중 대비 이탈 | ±5%p 이내 유지 | ±5%p 초과 시 원인 기록 | 지속 이탈 시 비중 조정 검토 |

이 프레임은 **분할 매수 완료 후** 적용하는 것이 원칙입니다.
재조정은 **분기마다 한 번**이 기본입니다.
월마다 상품을 바꾸면 거래비용만 늘고 수익률이 떨어집니다.

## 갈아타기 성공 판정 기준

D+90 시점에서 갈아타기의 성공 여부를 세 가지 지표로 판정합니다.

### 지표 1: 현금흐름 실현도
- 최근 수령 흐름이 월 154만원 목표에 근접 → 성공
- ±20% 초과 편차 → 포트폴리오 재조정 필요

### 지표 2: 생활비 버킷 소진 속도
- 생활비 버킷 0.84억이 예상보다 빠르게 줄고 있지 않은가
- 실제 월 생활비가 계획 350만원을 크게 웃돌지 않는가
- 국민연금 수령 전 공백기를 버틸 현금 여력이 유지되는가

### 지표 3: 심리적 안정
- 안산 생활에 적응했는가
- 월 현금흐름 수령이 실제로 "생활이 돌아간다"는 감각을 주는가
- 이 갈아타기를 **다른 사람에게 권할 수 있는가**

세 지표 모두 긍정적이면 **갈아타기 성공**.
하나라도 부정적이면 D+180(반년) 시점에서 재검토합니다.
