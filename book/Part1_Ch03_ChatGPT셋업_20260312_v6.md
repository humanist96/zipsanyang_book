# ChatGPT를 부동산 비서로 만들기

AI 도구를 비교하고 ChatGPT를 메인으로 선택했습니다.
이 챕터에서는 ChatGPT Plus 가입부터 Custom Instructions 설정, 그리고 첫 프롬프트 실행까지 완료합니다.
챕터가 끝나면 ChatGPT가 내 투자 조건을 기억하는 맞춤형 부동산 비서가 됩니다.

**학습 목표**
- ChatGPT Plus 가입과 GPT-4o 모델 선택을 완료합니다
- Custom Instructions에 나의 투자 조건과 답변 방식을 설정합니다
- 첫 프롬프트(구매력 계산)를 실행하여 설정이 제대로 되었는지 확인합니다

**핵심 키워드**
ChatGPT Plus / GPT-4o / Custom Instructions / Memory / 구매력 계산

## 셋업에 필요한 3가지 준비

본격적인 실습에 앞서, AI 투자 비서를 만들기 위한 최소한의 도구를 준비해야 합니다.

이 책의 모든 프롬프트는 무료 버전보다 훨씬 강력한 성능을 가진 **GPT-4o** 모델을 기반으로 합니다.

01. https://chat.openai.com 접속
02. 회원가입 후 로그인
03. 'Upgrade to Plus' 클릭

## ChatGPT 인터페이스 익히기

모든 준비가 끝났으면 ChatGPT 화면을 살펴보겠습니다. chat.openai.com에 접속하면 다음과 같은 화면이 나옵니다.

```
┌─────────────────────────────────────────────────────────┐
│  ☰ ChatGPT                                    [계정]    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  [+ 새 채팅]                                            │
│                                                         │
│  최근 대화 목록:                                         │
│  - 서울 아파트 분석                                      │
│  - 대출 계산 문의                                        │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│             대화 내용 표시 영역                           │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  [📎] [메시지 입력...]                          [전송↗] │
└─────────────────────────────────────────────────────────┘
```

| 버튼 | 위치 | 기능 |
|------|------|------|
| **+ 새 채팅** | 좌측 상단 | 새로운 대화 시작 |
| **☰ 메뉴** | 좌측 상단 | 설정, 대화 기록 접근 |
| **📎 클립** | 입력창 좌측 | 파일 업로드 (엑셀, PDF 분석 가능, 유료) |
| **계정** | 우측 상단 | 설정, 플랜 변경 |

파일 업로드 기능을 활용하면 "이 실거래가 데이터 분석해줘"라고 하면서 엑셀 파일을 올려 ChatGPT가 분석하도록 할 수 있습니다. 이 기능은 유료(Plus) 사용자만 이용할 수 있습니다.

## GPT-4o 모델 선택하기

ChatGPT에는 여러 모델이 있습니다. 이 책에서 사용하는 **GPT-4o**가 가장 강력합니다.

| 모델 | 특징 | 사용 조건 |
|------|------|----------|
| **GPT-3.5** | 기본 모델, 빠르지만 성능 제한 | 무료 |
| **GPT-4** | 강력한 분석, 상대적으로 느림 | Plus 필요 |
| **GPT-4o** | 최신 모델, 빠르고 강력 | Plus 필요 |

대화 창 상단의 모델 선택 드롭다운을 클릭하고 "GPT-4o"를 선택하면 됩니다.

💡 **Tip.** 무료 버전의 GPT-3.5로도 기본적인 질문은 가능합니다. 하지만 복잡한 계산이나 파일 분석은 GPT-4o가 훨씬 정확합니다. 예산이 허락한다면 Plus 구독을 추천합니다.

## Custom Instructions로 부동산 비서 만들기

ChatGPT에게 매번 "나는 무주택자이고, 예산은 얼마이고..."라고 설명하는 것은 번거로운 일입니다.

**Custom Instructions** — 한 번 입력해둔 정보가 모든 대화에 자동으로 적용되는 기능입니다. ChatGPT에게 미리 알려주는 자기소개라고 생각하면 됩니다. "나는 이런 사람이고, 이런 식으로 답변해줘"라고 설정해두면 매번 설명할 필요가 없어집니다.

### 설정 방법

01. 좌측 하단 [계정] 또는 우측 상단 프로필 클릭
02. [Settings] 선택
03. [Personalization] 탭 클릭
04. [Custom Instructions] 클릭
05. 두 개의 입력란에 내용 작성
06. [Save] 클릭

### 두 가지 입력란

| 입력란 | 영문 질문 | 작성 내용 |
|--------|----------|----------|
| **첫 번째** | "What would you like ChatGPT to know about you?" | 나에 대한 정보 (투자 조건) |
| **두 번째** | "How would you like ChatGPT to respond?" | 답변 방식 (전문가처럼, 표로 정리 등) |

```
- DTI: 40%
```

```
- 선호 평형: 전용 59㎡~84㎡

- 위험 성향: 보수적
```

### 두 번째 입력란: 답변 방식

```
## 역할
당신은 15년 경력의 부동산 투자 전문 어드바이저입니다.
서울/수도권 아파트 투자에 특화되어 있습니다.

## 답변 원칙
1. **수치 기반**: 모든 분석에 구체적인 숫자와 출처 포함
2. **규제 적용**: 2025년 10월 15일 규제 기준 적용
3. **시나리오 제시**: 보수적/중립/낙관 3가지 시나리오 제공
4. **리스크 명시**: 투자 리스크를 반드시 언급

## 출력 형식
- 핵심 결론을 먼저 말하고, 근거를 이어서 설명
- 복잡한 정보는 표로 정리
- 계산이 필요하면 단계별로 보여주기
- 전문 용어 사용 시 괄호 안에 설명 추가

## 피해야 할 것
- 막연한 낙관론 ("오를 것 같습니다")
- 출처 없는 수치
- 법률/세금 자문 (전문가 상담 권유)
```

💡 **Tip.** 규제 수치(LTV, DSR 등)는 Custom Instructions에 넣어두면 매번 입력하지 않아도 됩니다. 다만, 규제가 변경되면 반드시 업데이트하세요. 이 책 기준 최신 규제는 2025년 10월 15일자입니다.

## 설정 전후, 답변이 이렇게 달라집니다

같은 질문을 해도, Custom Instructions 설정 여부에 따라 답변이 완전히 달라집니다.

**설정 전 (기본 상태):**
```
질문: "내가 살 수 있는 아파트 가격은?"

ChatGPT: 아파트 구매 가능 가격은 개인의 재정 상황, 대출 조건,
지역 등 여러 요소에 따라 달라집니다.
일반적으로 LTV(담보인정비율)는 40~70%이며,
DTI(총부채상환비율)는 40~60%가 적용됩니다.
정확한 계산을 위해 은행 상담을 권장합니다.
```
→ **일반론만 나옵니다.** 구체적 계산이 없습니다.

**설정 후 (투자 조건 입력됨):**
```
질문: "내가 살 수 있는 아파트 가격은?"

ChatGPT: 홍길동님의 조건으로 계산해드리겠습니다.

| 항목 | 계산 | 금액 |
|------|------|------|
| 스트레스 DSR 기준 대출 | 연소득 × DSR / (금리+1.5%) | 약 2.6억 |
| 실제 대출 가능액 | LTV·DSR 중 낮은 값 | 약 2.6억 |
| 최대 매수 가능가 | 현금 + 대출 | 약 5.1억 |

추천 예산: 안전 마진 고려 시 4억 5천만 원 이하
```
→ **맞춤형 계산과 구체적 금액이 나옵니다.** 이것이 Custom Instructions의 힘입니다.

한 번 설정해두면 모든 대화에 적용되기 때문에, 매번 설명할 필요 없이 바로 맞춤 분석을 받을 수 있습니다.

## Memory 기능으로 대화 맥락 유지하기

ChatGPT에는 **Memory** 기능도 있습니다. Custom Instructions는 내가 미리 알려주는 정보이고, Memory는 대화하면서 ChatGPT가 스스로 기억하는 정보입니다.

| 기능 | 특징 | 예시 |
|------|------|------|
| Custom Instructions | 사용자가 직접 입력 | "나는 31세 무주택자" |
| Memory | 대화 중 자동 저장 | "지난번에 하남시 관심 있다고 했습니다" |

예를 들어 "지난번에 하남시 분석했었는데, 광명시랑 비교해줘"라고 하면, ChatGPT가 이전 대화 내용을 바탕으로 답변해줍니다.

Memory 확인 및 관리 방법은 Settings → Personalization → Memory에서 할 수 있습니다. ChatGPT가 기억한 항목 목록을 확인하고, 불필요한 항목은 삭제할 수 있습니다.

> ⚠️ **주의:** Memory에 잘못된 정보가 저장되면 이후 답변에도 영향을 미칩니다. 투자 조건이 바뀌었다면(예: 관심 지역 변경, 자금 규모 변동) Memory도 함께 정리해야 합니다.

Custom Instructions를 바꿔도 이전 대화에는 적용되지 않습니다. 설정 변경 후에는 새 채팅을 열고 시작하세요. 여러 개의 Custom Instructions를 저장하고 싶다면, **GPTs (Custom GPT)**를 활용하여 '부동산 투자용 GPT', '일상 대화용 GPT'를 분리할 수 있습니다.

## 내 구매력 계산하기

모든 설정이 끝났습니다. 이제 설정이 제대로 되었는지 **첫 프롬프트를 실행**해서 확인해 보겠습니다.

### 프롬프트를 보내기 전에

가장 기본적인 질문은 "내가 얼마짜리 집을 살 수 있는가"입니다. Custom Instructions에 이미 투자 조건을 입력했기 때문에, 별도로 조건을 말하지 않아도 ChatGPT가 알아서 적용합니다.

01. ChatGPT에서 새 대화창(New Chat)을 엽니다.
02. GPT-4o 모델이 선택되어 있는지 확인합니다.
03. 아래 프롬프트를 복사해서 붙여넣습니다.

### 프롬프트

> **프롬프트**
>
> 나의 투자 조건으로 구매 가능한 최대 아파트 가격을 계산해줘.
>
> 다음 항목을 포함해서 분석해줘:
> 1. 대출 가능 금액 (LTV, 스트레스 DSR 적용)
> 2. 최대 매수 가능가
> 3. 안전 마진 고려한 추천 예산
> 4. 부대비용 (취득세, 중개수수료 등)
> 5. 주의사항
>
> 2025년 10월 15일 규제 기준으로 계산해줘.

Custom Instructions를 설정하지 않았다면, 프롬프트 앞에 자신의 조건(연소득, 가용현금, 주택 보유 수, 기존 대출 등)을 추가해야 합니다.

### AI 응답 확인하기

Custom Instructions에 완성형 예시(가용현금 2.5억, 연소득 6,500만원, 무주택자)를 입력했다면 다음과 유사한 결과가 나옵니다.

```
## 입력 조건 (Custom Instructions 기준)
- 가용 현금: 2억 5천만 원
- 연소득: 6,500만 원
- 주택 보유: 무주택자
- 기존 대출: 없음

## 대출 가능 금액

### LTV 기준 (담보인정비율)
- 적용 LTV: 40% (2025.10.15 규제)
- LTV 기준 대출 = 매매가 × 0.4

### 스트레스 DSR 기준
- 기준 금리: 4.5% (가정)
- 스트레스 금리: 4.5% + 1.5% = 6.0%
- DSR 한도: 40%
- 연간 원리금 상환 가능액: 6,500만 원 × 40% = 2,600만 원
- 30년 만기 기준 대출 가능액: 약 2억 6천만 원

## 최대 매수 가능가

| 구분 | 금액 | 비고 |
|------|------|------|
```

이와 유사한 결과가 나오면 Custom Instructions 설정이 성공한 것입니다. 결과에서 확인할 포인트는 다음과 같습니다.

| 확인 항목 | 기준 |
