# 실습 ㉘: 최종 의사결정 보고서

Part 2부터 시작된 긴 여정이 마지막 실습에 도달했습니다.
거시경제 분석, 미시경제 분석, 정책 분석, 투자 전략 수립, 지역 선정, 단지 비교, 적정가 분석, 임장과 재평가까지 수많은 데이터와 분석을 거쳤습니다.
이 실습에서는 지금까지의 모든 분석을 종합하여 "이 집을 사도 되는가"에 대한 최종 판정을 내립니다.
여기서는 결론을 서두르기보다, 마지막으로 한 번 더 브레이크를 밟는다는 마음이 중요합니다. 매수 직전의 사람은 확신과 불안이 동시에 커지기 쉬우므로, 오히려 이 시점일수록 기준을 더 단순하고 보수적으로 보는 편이 안전합니다.
많은 분이 이 단계에 오면 이미 마음속 답을 정해 둔 채 보고서를 쓰고 싶어집니다. 그래서 더더욱, 이번 실습은 등을 떠미는 문서가 아니라 한 번 멈춰 세우는 문서라고 생각하는 편이 좋습니다.

**학습 목표**

- Part 2~8까지의 모든 분석 결과를 한 장의 보고서로 종합합니다
- 4가지 판정 기준(재무적 안전성, 시장 타이밍, 리스크 수용, 목표 달성 가능성)으로 매수 가능 여부를 판단합니다
- 출구 전략(목표 매도, 손절 기준, 시장 변화 대응)을 사전에 설정합니다

**핵심 키워드**
최종 판정 / 재무적 안전성 / 시장 타이밍 / 리스크 시나리오 / 출구 전략 / 행동 가이드

## 매수 버튼을 누르기 전 마지막 점검이 필요하다

부동산은 인생에서 가장 큰 쇼핑 중 하나입니다.
잘못 사면 몇 년간 발이 묶이거나, 손해를 보고 팔아야 합니다.
그래서 최종 의사결정 보고서를 통해 마지막으로 한 번 더 점검합니다.

수억 원짜리 계약서 앞에 서면 "혹시 상투 잡는 건 아닐까?"라는 불안감이 생기기도 하고, 반대로 "지금 아니면 평생 집 못 산다"는 조급함에 휩쓸리기도 합니다.
사람은 **확증 편향**(보고 싶은 것만 보는 심리)이나 손실 회피 편향에 빠지기 쉽기 때문입니다.

AI는 감정이 없습니다.
오직 숫자와 논리로만 판단합니다.
의사결정 보고서는 확신을 뒷받침하는 근거가 되거나, 위험한 투자를 막아주는 마지막 브레이크 역할을 합니다.
결론이 마음에 들지 않더라도 그 자체가 중요한 신호일 수 있습니다. 좋은 보고서는 내 판단을 예쁘게 포장하는 문서가 아니라, 불편한 점을 드러내는 문서이기 때문입니다.

## 4가지 판정 기준으로 매수 가능 여부를 확인하는 방법

AI가 최종 판정을 내릴 때 사용하는 4가지 기준입니다.
이 항목들은 외워야 할 시험 범위라기보다, 계약 직전 순서대로 다시 확인할 체크포인트라고 생각하면 됩니다.

| 기준                 | 질문                                    | 판정 기준                    |
| -------------------- | --------------------------------------- | ---------------------------- |
| **재무적 안전성**    | 월 상환액이 월 소득 대비 무리 없는가?   | 30% 초과 시 → 보류 또는 철회 |
| **시장 타이밍**      | 현재 사이클 위치가 매수에 적합한가?     | 고점 근접 시 → 보류          |
| **리스크 수용**      | 최악의 시나리오를 감당할 수 있는가?     | 감당 불가 시 → 철회          |
| **목표 달성 가능성** | 투자 목적과 보유 계획이 현실적으로 맞는가? | 목표와 계획이 어긋나면 → 보류 |

4가지 기준 중 **하나라도 탈락이면 매수 불가**입니다.
4가지 모두 통과해야 매수 또는 조건부 매수가 됩니다.

> ❓ **궁금해요 출구 전략이란?**
>
> **출구 전략** — 매수 전에 "언제, 어떤 조건에서 팔 것인가"를 미리 정해두는 것입니다.
> 목표 수익률에 도달하면 매도하고, 손실이 일정 한도를 넘으면 손절하며, 시장 환경이 크게 변하면 재평가합니다.
> 들어갈 때 나올 계획이 있어야 안전합니다.

## 판정 규칙

1. **4가지 기준 전부 통과**: 재무적 안전성, 시장 타이밍, 리스크 수용, 목표 달성 가능성 중 하나라도 탈락이면 매수 불가
2. **재무적 안전성**: `예상 월 상환액 / 월 가구 실수령 소득`을 기준으로 판단하세요.
   - 30% 초과면 기본적으로 `보류` 또는 `철회`
   - 두 값 중 하나라도 없으면 재무적 안전성은 `확인 필요`로 표시하고, 최종 판정은 `매수`/`조건부 매수` 대신 `보류` 또는 `철회`로만 제시하세요.
3. **리스크 수용**: 최악 시나리오에서 손실·상환 부담·보유 스트레스를 감당하기 어렵다면 `철회` 또는 `보류`

## 분석 규칙

4. **판정 근거 5가지**: 최종 판정에는 반드시 5가지 명확한 근거 제시
5. **시나리오 2개**: 기본 시나리오 + 최악 시나리오를 반드시 제시하되, 확률은 `%` 대신 `낮음 / 보통 / 높음`으로 제시
6. **출구 전략 필수**: 목표 달성, 손절, 시장 변화 3가지 출구 시나리오 제시
7. **임의 기준 도입 금지**: 이 프롬프트에 없는 일반 기준(예: `DTI`, `DSR 40%`, `LTV xx%`, 일반 손절률)을 새로 도입하지 말고, 반드시 사용자 입력과 본문에 제시된 4가지 판정 기준만 사용하세요.

## 행동 규칙

8. **트리거 조건**: 조건부 매수 시 구체적 트리거 조건 + 타임라인 명시
9. **비용 포함**: 매수가 외 취득세, 중개수수료, 대출이자를 고려한 실제 필요 자금을 다루되, 세율·보수율·금리가 입력되지 않았으면 `실제 필요 자금 = 매수가 + 취득세(확인 필요) + 중개수수료(확인 필요) + 이자(확인 필요)`처럼 계산식만 제시하고 원화 합계는 만들지 마세요.
10. **숫자 생성 금지**: 입력되지 않은 소득, 상환액, 세율, 미래 가격, 금융 심사 기준, 총비용 합계는 임의로 만들지 말고 `확인 필요` 또는 계산식/범위로 제시하세요.

**리스크 시나리오**는 기본(가능성 `보통`)과 최악(가능성 `낮음`)으로 나뉩니다. 기본 시나리오에서는 완만한 상승 또는 보합, 최악 시나리오에서는 GTX 연기와 금리 재상승으로 가격 조정과 보유 부담 확대가 동시에 올 수 있다는 분석이 제시됩니다.

**출구 전략**은 4가지 시나리오로 구분됩니다.

| 시나리오       | 조건                     | 행동                           |
| -------------- | ------------------------ | ------------------------------ |
| **목표 달성**  | 7억 이상 도달 (27% 상승) | 매도 검토                      |
| **부분 수익**  | 6억 도달 (9% 상승)       | 보유 지속 vs 매도 재평가       |
| **손절**       | 4.5억 이하 (18% 하락)    | 매도 후 재투자 검토            |
| **GTX 개통**   | 2030년 개통 확정         | 단기 급등 시 일부 차익 실현    |

## AI 판정 이후 반드시 직접 확인해야 할 5가지

AI가 "매수"라고 판정했더라도, 아래 5가지를 반드시 직접 확인해야 합니다.
하나라도 체크되지 않은 항목이 있다면, AI 판정과 무관하게 보류하는 것이 안전합니다.

| 체크 항목                         | 확인 내용                          | AI가 판단할 수 없는 이유                            |
| --------------------------------- | ---------------------------------- | --------------------------------------------------- |
| 월 상환액이 감당 가능한가?        | 월 상환액 < 소득 30%               | AI가 금리 변동을 과소평가할 수 있음 → 은행에서 직접 확인 |
| 최악의 상황을 버틸 수 있는가?     | 손실 한도 내 감당 가능             | AI 최악 시나리오가 실제 최악이 아닐 수 있음          |
| 가족(배우자) 동의를 받았는가?     | 동의 완료                          | AI가 판단할 수 없는 영역                             |
| 대출 승인 가능성 확인했는가?      | 은행 사전 상담 완료                | AI가 개인 신용 상황을 알 수 없음                     |
| 비상 자금이 있는가?               | 6개월 생활비 확보                  | AI가 생활비를 과소 추정할 수 있음                     |

AI는 입력된 데이터 기반으로만 판단하며, 개인 사정은 알 수 없습니다.
AI 판정은 "참고"이며, 최종 결정은 본인이 내리는 것입니다.

> ⚠️ **주의:** AI가 "매수"라고 했다는 이유만으로 바로 계약을 진행하면 안 됩니다. AI는 조언자이고, 결정권자는 본인입니다. 위 5가지 항목을 모두 확인한 뒤에 행동으로 옮깁니다.

💡 **Tip.** 이 실습에서는 4인 가족(갈아타기) 기준으로 의사결정 보고서를 작성했습니다. **신혼부부**라면 투자 유형을 "레버리지형"으로, 정책 혜택에 생애최초 조건을 입력하세요. **은퇴 준비**라면 투자 유형을 "안정형"으로, 출구 전략에 다운사이징 시나리오를 추가하면 됩니다.

## 핵심 정리

- **전체 분석 종합**: Part 2~8까지의 모든 분석 결과를 한 장의 보고서로 요약하여, 일관된 논리 위에서 최종 판단을 내립니다.
- **4가지 판정 기준**: 재무적 안전성, 시장 타이밍, 리스크 수용, 목표 달성 가능성 중 하나라도 탈락이면 매수가 불가합니다.
- **출구 전략**: 들어갈 때 나올 계획을 미리 세워두면, 시장이 어떻게 변하든 대응할 수 있습니다.
- **최종 결정은 본인의 몫**: AI는 냉철한 분석 파트너이지만, 최종 결정은 언제나 본인이 내립니다.
