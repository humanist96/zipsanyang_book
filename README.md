## 목차

### Part 0. 부동산 투자, AI로 시작합니다

- 부동산 투자에 AI가 필요한 시대
  - 투자는 선택이 아니라 필수입니다
  - 부동산이 안전자산으로 불리는 이유
  - 아무 부동산이나 사면 다 오르는 시대는 끝났습니다
  - 데이터를 직접 검증하는 시대입니다
- 부동산이 어렵게 느껴지는 이유
  - 부동산은 '비교 학문'입니다
  - 부동산은 '종합 예술'입니다
- 그래도 부동산을 알아야 하는 이유
- 저평가 매물을 찾는 가치 투자
- 거주 가치와 투자 가치, 둘 다 봐야 합니다
- 가격이 오르는 아파트의 조건
- 부동산 가격에 영향을 끼치는 요인
- 부동산 거래 방식이 이렇게 바뀌었습니다
- 부동산과 기술의 결합, 프롭테크
  - 해외 사례: Zillow
  - 국내 프롭테크의 성장
- 누구나 정보를 볼 수 있는 데이터 민주화 시대
- 프롭테크로 데이터 맛보기: 부동산 지인
  - 시장 강도 확인하기
  - 사분면 분석: 부동산의 봄, 여름, 가을, 겨울
  - 멀티차트로 흐름 파악하기
  - 거래량과 수요/공급 확인하기
- 프롭테크가 해결하지 못하는 것
- 같은 데이터를 보고도 결과가 다릅니다
  - 실제 사례: GTX-A와 두 투자자
- Zillow의 충격적인 실패: AI도 틀릴 수 있습니다
  - 실패의 3가지 원인
- AI가 가져다주는 4가지 실질적 이득
  - 시간: 임장 시간이 확 줄어듭니다
  - 비용: 교통비, 교육비가 줄어듭니다
  - 에너지: 꾸준히 공부할 수 있는 힘이 생깁니다
  - 정보: 여러 사이트를 한 번에 분석합니다
- AI와 프롭테크의 역할 분담
- AI를 도구로 쓰되, 맹신하지 않는 원칙
